# RabbitMQ 的安装-Docker

使用 Docker 安装起来更方便点

<!-- more -->

## 安装方法

[官网](https://www.rabbitmq.com/download.html) 有这样一段：

```shell
latest RabbitMQ
docker run -it --rm --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3.11-management
```

安装比较简单，这里就不演示了。

注意 Windows 下也可以使用 Docker，感兴趣的读者可以试试。
