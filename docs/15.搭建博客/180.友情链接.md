# 友情链接

本文介绍下友情链接。

‍

## 简介

友情链接也称为友链、交换链接、互惠链接、互换链接等，是具有一定互补优势的网站之间的简单合作形式，即分别在自己的网站上放置对方网站的 LOGO 或网站名称并设置对方网站的超级链接，使得用户可以从合作网站中发现自己的网站，达到互相推广的目的。

本站的友链地址为：[https://www.peterjxl.com/About/friend-link](https://www.peterjxl.com/About/friend-link)

可以使用 Voding 主题的 [图文卡片](https://doc.xugaoyi.com/pages/d0d7eb/#普通卡片列表) 来展示友链，效果：

## 自动化

传统的添加友链方式：先添加对方的友链信息，然后联系作者发送自己的网站信息。

作者收到后，再手动添加你的友链信息。

但这样就有点慢了，而且需要手工维护 —— 对于程序员来说当然是不必要的，我们可以通过程序来帮我们自动化维护。

方法有很多，就不一一说明了，可参考：

- 虹墨空间站（Twikoo 作者）：[通过 Vika 表格维护友链](https://www.imaegoo.com/2024/http-get-data-from-wps/)，支持通过 API 增、删、改、查表格中的数据
- [LiuShen's Blog | Github Action 实现友链状态检测](https://blog.liushen.fun/posts/c2262998/)
- [無名小栈 | 如何实现一个自动化友链流程](https://blog.imsyy.top/posts/2024/0521)

‍

## 日常维护

友链是需要维护的，不是说加了就完事了。

就我这几年的博客经验来说，见过太多的博客倒闭跑路了。这时我们需要及时移除友链 —— 因为如果搜索引擎发现你的博客有死链的话，会降低权重的。

如果每次都自己逐个打开友链，检查是否失效，就未免太麻烦了，建议使用自动化程序。

也可以用一些链接检查器：

- [Link Checker](https://link-checker.itea.dev/)：网站链接检查器，输入网站链接，找出其中的死链接，[代码开源](https://github.com/isixe/LinkChecker)。
- [死链检测器](http://www.deadlinkchecker.com/)：输入网站 URL，一键检测全站的内链和外链是否 404。
