# 博客正式迁移到 VitePress 啦

全面升级，欢迎来访~

‍<!-- more -->

## 什么是 VitePress

![VitePress](https://image.peterjxl.com/blog/VitePress-20260108130117-0hu242g.png)

VitePress 是 Vue 官方开发的框架，能快速生成静态网站。

详看官方介绍：[VitePress 是什么？](https://vitepress.dev/zh/guide/what-is-vitepress)

之前我曾详细介绍过如何用 VuePress 搭建个人博客（详见《[最全面的博客搭建教程](/Blog/Introduce/)》）。

而 VitePress 是基于 VuePress 的，但 VitePress 更强大，使用体验更棒。

‍

## 为什么迁移到 VitePress

最直接的原因是：效率。

随着时间的推移，我的博客文章越来越多（1000+），博客的编译、构建所需时间也越来越多。

后来有不少博友推荐我用 VitePress，最近试了下，发现确实很快。

以我自己的博客为例，测试结果如下：

|          | VuePress | VitePress |
| :------: | :------: | :-------: |
| 编译运行 |  340 秒  |   40 秒   |
| 构建打包 | 32 分钟  |  240 秒   |


此外，我个人感觉，访问速度也有很大的提升，切换文章也很丝滑。

VitePress 的热更新功能也很棒，比 VuePress 的好用多了，日常维护博客不用频繁地重启服务。

‍

## VitePress 的缺点

当然，VitePress 也有缺点，就是生态还不太完善：

- 社区：VuePress 作为老牌框架，用户数量大 ，文档、主题和插件丰富，大部分问题都有解决方案。而 VitePress 就没这么好了。
- 主题：目前主题的数量还是比较少的。
- 插件：我想添加 RSS，但没找到合适的插件，目前还是手工维护。

毕竟是新生框架，还需要时间沉淀。

但瑕不掩瑜，光凭节省大量时间这一点，我就决定要用它了（虽然迁移用了不少时间）。

我目前用的是博友开发的 Teek 主题，功能很我用的 vdoing 主题类似，且功能更多（有了这个，更下定决心要迁移了）。

此外，随着时间的发展，VitePress 一定会越来越好的，毕竟连官方都宣布将长期维护 VitePress，而 VuePress 已移交给社区团队了。

‍

## 最后

按理来说，迁移到 VitePress 后，我之前的博客也应该同步修改...

可惜个人精力有限，如果后续有时间再更新吧。而且 VuePress 和 VitePress 非常相似，很多配置都可以照搬过来，不必担心博客内容过时了。

这里推荐几个我觉得不错的入门文档：

- [VitePress 官方文档](https://vitepress.dev/zh/)
- [Vitepress Theme Teek 官方文档](https://vp.teek.top/)
- [One](https://teek.onedayxyy.cn/)：群里很活跃的大佬，有很多 VitePress 的优化配置，还有视频讲解
- [Hyde Blog](https://teek.seasir.top/)：作者写了很多 VitePress 的优化配置，我参考了很多

‍

目前，本博客还没完全迁移完，有部分功能还不能用（例如搜索），但基本功能已经完善，欢迎来访！
