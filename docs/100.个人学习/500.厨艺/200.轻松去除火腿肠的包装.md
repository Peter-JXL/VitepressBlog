# 轻松去除火腿肠的包装

偶尔吃泡面的时候会来几根火腿肠，但之前对于火腿肠包装的去除一直不得要领，实在烦了就去特地学习了下，笔记如下。
‍<!-- more -->
‍

准备一根牙签从顶部插进去（用剪刀或者其他东西夹住顶部也可以）：

![](https://image.peterjxl.com/blog/image-20240823203454-oxw3fo3.png)​
‍

转上几圈，然后一扯。就可以把包装线去除了：

![ham-sausage-pack-remove](https://image.peterjxl.com/blog/ham-sausage-pack-remove-20240823203800-ixdixc1.gif)​
‍

然后就可以轻松将剩余部分剥下来。
‍
