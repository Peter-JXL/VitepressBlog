## 应该学习 Docker 吗？

对于计算机行业来说，非常推荐学习 Docker，它非常流行，功能强大，能很好地解决环境问题。

对于其他行业来说，也可以考虑学习。通过 Docker，可以很方便地使用各种软件，而不用担心环境问题。例如很多人在使用的 NAS，就可以用 Docker 安装很多很多软件。

此外，本系列教程通俗易懂，学习起来很容易，多学点总不是坏事。
