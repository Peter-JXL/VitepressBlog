# 免费开源的 PDF 处理神器：Stirling-PDF

![](https://image.peterjxl.com/blog/image-20240919172235-sjlvivi.png)​

PDF 是工作中常见的文件格式，经常需要对其进行合并、分割、旋转、加水印等操作。而目前市面上大部分 PDF 软件，免费版只能用来看 PDF，修改 PDF 的功能都是付费的。

今天就和大家分享一个本地化处理 PDF 的神器：Stirling-PDF，它功能强大，几乎可以满足任何操作，而且是本地运行，不用担心数据泄漏的问题。

项目地址：[https://github.com/Stirling-Tools/Stirling-PDF](https://github.com/Stirling-Tools/Stirling-PDF)，截止到 2024-9 月已经有了 42k 的 star！

## 项目简介

`Stirling-PDF`​ 是一个基于 Web 的开源 PDF 处理工具，功能非常丰富：

![](https://image.peterjxl.com/blog/image-20240919174716-dkpyv0m.png)​


性能特色：

* 功能多：可以合并、拆分、加解密、旋转、压缩...  还可以搜索功能
* 效率高：使用了高效的算法，处理速度快，支持批量操作
* 跨平台：基于 Docker，Windows、Mac、Linux 都能使用
* 界面美：界面简洁、明了，支持夜间模式
* 多语言：目前支持 38 种国家和地区的语言

## 项目安装

该项目基于 Docker，如果你有使用的话可以一条命令完成运行：

```dockerfile
docker run -d \
  -p 8080:8080 \
  -v ./trainingData:/usr/share/tessdata \
  -v ./extraConfigs:/configs \
  -v ./logs:/logs \
  -e DOCKER_ENABLE_SECURITY=false \
  -e INSTALL_BOOK_AND_ADVANCED_HTML_OPS=false \
  -e LANGS=en_GB \
  --name stirling-pdf \
  frooodle/s-pdf:latest
```
‍
然后访问 `http://服务器IP:8080`​，即可看到主页面，然后就可以愉快地使用了：

![](https://image.peterjxl.com/blog/image-20240919184003-ka7862z.png)​

除了 Docker 之外，也可以去 [release](https://github.com/Stirling-Tools/Stirling-PDF/releases) 页面下载 exe 文件进行安装：

![](https://image.peterjxl.com/blog/image-20240919185432-00awn9f.png)​


还可以设置登录密码，避免被其他人误用（具体操作请看官网文档）。

![](https://image.peterjxl.com/blog/image-20240919191331-93i0pj3.png)​


## 使用

使用起来也非常简单，选择想要的功能，上传 PDF 文件就可以了，例如单击「旋转 PDF」后，就会跳转到上传页面，上传后就可以进行旋转：

![](https://image.peterjxl.com/blog/image-20240919184306-xrsb1a3.png)​


如果你要对 PDF 进行多种操作，例如旋转、合并、重新排列... 那么逐个打开功能并操作也是很麻烦的，因此我们可以用「PDF 多功能工具」，打开 PDF 后可以对每个页面进行更细致的操作：

![](https://image.peterjxl.com/blog/image-20240919184423-k2ixskr.png)​


## 移动端使用

如果你安装的远程服务器上，还可以在手机上使用浏览器访问，然后对其进行操作。

如果是自己家的电脑，也可以通过局域网来使用：

![](https://image.peterjxl.com/blog/image-20240919185127-v5dxw6v.png)​


注意 IP 的问题：先在电脑上用「ipconfig」看一下 IPv4 地址，然后在手机/平板上用浏览器访问「IP: 8080」
