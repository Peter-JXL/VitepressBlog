# ImTip：又一款小巧的输入法状态提示工具

​![](https://image.peterjxl.com/blog/image-20241009104702-o4ttmnh.png)​

ImTip 是一款小巧的 Windows 工具，用来在有输入光标的地方，显示输入法状态。

官网：[https://imtip.aardio.com](https://imtip.aardio.com)

可以在官网或我的网盘里下载：https://pan.quark.cn/s/7bad5be67ea5


‍<!-- more -->

## 介绍

一款显示输入法状态的工具。特点：

* 仅 660 KB
* CPU 占用极低
* 开源在：[https://github.com/aardio/ImTip](https://github.com/aardio/ImTip)，目前已经有 1.5k 的 star
* 单文件绿色软件，无任何外部依赖
* 兼容 XP，Vista，Win7，Win8，Win10，Win11 ……
* 理论上支持所有输入法，几乎支持所有软件
‍
## 效果

ImTip 通过在输入光标处显示 2 个简单的图标 —— 提前知道中英、中英标点、全半角、大小写、多语言键盘布局等所有状态。

## 有什么用？

这时候可能有人会问了，这玩意有什么用？这里重复下上篇文章的观点：

1. 方便。对于我来说，平时基本都是用盲打的，打字过程中还要低头去看指示灯是一件麻烦事。
2. 有了 ImTip 就可以关掉输入法自带的状态栏，屏幕更干净。
‍
网友这样说：

> [二零零九不是质数](https://space.bilibili.com/67682364)：我不同意其他评论的意见，我觉得很有用，我这里有三个显示屏，要看状态栏一眼还要扭头去看，而且操作的软件用快捷键需要在英文状态下，有时候就会经常忘记，打了几个字母才意识到又要删掉按 shift 再重打，这个软件非常能帮到我
>
> [傲世_豪情](https://space.bilibili.com/67918494)：如果你有一块 27 英寸以上的屏幕，或者多块屏幕就知道有多好用了。如果一个文字工作者每次都要先敲两个字来确定当前输入状态那他的效率可太低了
‍
## 缺点

InputTip 的作者这样[点评](https://meta.appinn.net/t/topic/60614)：

> imTip 是我早期使用的软件，该软件的最大问题是稳定性特别差，突然的功能失效，突然的奇怪错误报告，突然的自动退出，这些太过于折磨，这是在我多台电脑上的表现。并且 imTip 尚有很多界面不支持，例如微信界面，Jetbrains IDE（需要额外设置）等。
‍
此外还是有不少地方不适配，UI 会有偏移的问题。


## 最后

总得来说还是不错的一款软件，小巧强大，也不会占用很多资源，界面也还 OK（感觉比 InputTip 好看点），值得一试。

在 Windows 下，类似的工具还有很多，就不再单独写文章介绍了，这里简单说下：

* KBLAutoSwitch：开源，中英文输入法自动切换，输入法指示器，配置项丰富
* CapsCursor：中文输入法鼠标尾巴
