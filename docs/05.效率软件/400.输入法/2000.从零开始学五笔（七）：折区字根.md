# 2000.从零开始学五笔（七）：折区字根

讲解五笔中折区有什么字根和规律

<!-- more -->

讲解格式：

* 先介绍按键的区位号、口诀内容、口诀说明
* 然后列每个字根能组成什么汉字，难拆字将用中括号标出
‍
### N 键

区位号：51

口诀：已半巳满不出己，左框折尸心和羽

说明：

* 已半巳满不出己：指的是“已”，“巳”，“己”三个字的不同之处，“已”字半封口，“巳”字全封口
* 左框：之前说过右框“匚”，左框就是左右相反

  ![](https://image.peterjxl.com/blog/image-20241023203006-n8kv4tz.png)​
* 折：指的是笔画“折”，因为 N 键是折区的第一个按键
* 尸：一个字根，还有个变体

  ![](https://image.peterjxl.com/blog/image-20241023202814-6fz74hn.png)​
* 心：一个字根，还有变体竖心旁“忄”，变体：

  ![](https://image.peterjxl.com/blog/image-20241023202852-9h81vu7.png)​
* 羽：一个字根
‍
示例汉字：

* 已：唯一一个不能组成字的字根
* 巳：导，包，异，港，撰，
* 己：起，记，改，纪，凯
* 左框：巨，追，所，候，夬，快，决，[鹿]
* 折：乙，甩，书，与，瓦，片，乜，鸟
* 尸：民，展，尺，尽，户
* 尸的变体：声，馨，眉，媚，嵋
* 心：想，意，感，思，必
* 忄和变体：性，情，快，恭，添
* 羽：翻，翼，耀，翰，廖
‍
### B 键

区位号：52

口诀：子耳了也框向上

说明：

* 子：一个字根
* 耳：一个字根，还有变体左耳旁“阝”，变体有两个：

  ![](https://image.peterjxl.com/blog/image-20241023203715-cbggr9o.png)​
* 了：一个字根
* 也：一个字根
* 框向上：一个开口朝上的方框，凵
* 巜：两折，隐藏字根
‍
示例汉字：

* 子：学，好，教，存，孙，[孑]
* 耳：最，取，闻，敢，聂
* 阝：卫，报，却，节，迎
* 阝变体 1：卫，报，却，节，迎
* 阝变体 2：创，范，顾，卷，危
* 了：疗，辽，亨，哼，亟，[矛，予]
* 也：他，她，地，施，拖
* 凵：凶，出，函，离
* 巜：粼


### V 键

区位号：53

口诀：女刀九臼山朝西

说明：

* 女，刀，九，臼，都是字根
* 山朝西：值得是“山”这个字逆时针转 90 °，变成“⺕”
* 巛：三折，隐藏字根
‍
示例汉字：

* 女：好，如，她，数，安
* 刀：分，解，照，切，超，[那]
* 九：丸，究，热，杂，仇
* 臼：搜，插，陷，毁，鼠
* ⺕：雪，尹，庚，兼，肃
* 巛：巡，巢，瑙，淄，邋
‍
### C 键

区位号：

口诀：又巴马，丢矢矣

说明：

* 又，巴，马，都是字根
* 丢矢矣：值得是“矣”字丢掉矢，变成“厶”
* “又”有两个变体：

​![](https://image.peterjxl.com/blog/image-20241023205851-3xl9icp.png)​

其实这个变体也很好理解，变体的第一笔是折，所以是第五区，第二笔是捺，所以在折区第四个按键  C 上。


示例汉字：

* 又：对，度，报，受，变
* 巴：把，色，肥，爬，绝
* 马：吗，验，骂，冯，闯
* 厶：能，么，公，治，即，[离]，注意“偶”字不包含该字根
* 又变体 1：通，勇，通，疑，凝，注意“专”字不包含该字根
* 又变体 2：经，轻，劲，茎，氢
‍
### X 键

区位号：55

口诀：慈母无心弓和匕，幼无力

说明：

* 慈母无心：指的是“母”这个字去掉中间的部分，以及其变体：

  ​![](https://image.peterjxl.com/blog/image-20241025154326-u7fvd9d.png)​
* 弓：一个字根
* 匕：一个字根，有个变体为 ![](https://image.peterjxl.com/blog/image-20241023210325-i8i0og6.png)​
* 幼无力：指“幼”字去掉右边的“力”字，变成“幺”。变体：绞丝旁“纟”和![](https://image.peterjxl.com/blog/image-20241023210448-j77s6di.png)​


示例汉字：

* 慈母无心：母，每，毒，敏，繁，贯，惯，毋
* 慈母无心的变体：互，缘，蠡
* 弓：第，强，张，弱，弗
* 匕和变体：能，比，老，它，死，[曳]
* 幺：系，素，率，幻，玄
* 纟和变体：经，给，线，丝，乡


## 建议

不要单独背口诀，要和字根的分布规律一起记忆。

例如，“慈母无心”这个字根第一笔是折，第二笔也是折，所以在折区最后一个按键上。
