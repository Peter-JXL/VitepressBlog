# 从零开始学双拼（二）：全面开启双拼

简单说说各种设备怎么开启双拼

‍<!-- more -->

## iOS

其实添加双拼键盘，和添加五笔键盘差不多，具体可以看《[启用 iPhone 原生的五笔输入](https://www.peterjxl.com/typewriting/Wubi-iPhone/)》，这里就不赘述了

你也可以安装流行输入法，然后切换为双拼。
‍
## Android

貌似没有原生的双拼，可以安装下主流的输入法，然后切换为双拼。

上一篇博客已经介绍了不少输入法，也可以看我的另一篇文章《[十款好用的输入法软件推荐](https://www.peterjxl.com/typewriting/software-recommend/)》。
‍
## macOS

目前也已经支持原生双拼了，在右上角可以打开键盘偏好设置：

​![](https://image.peterjxl.com/blog/image-20241107172316-z20hnwv.png)​
‍

然后点击左下角的加号，添加新键盘：

​![](https://image.peterjxl.com/blog/image-20241107172355-epq0w99.png)​
‍

选择简体中文，然后添加简体双拼：

​![](https://image.peterjxl.com/blog/image-20241107172453-4xkrdl2.png)​
‍

添加后，即可选择双拼方案：

​![](https://image.peterjxl.com/blog/image-20241107172504-rvasuvp.png)​
‍

## Windows

按下「Win + I」打开设置，选择「时间和语言」：

​![](https://image.peterjxl.com/blog/image-20241107101507-1zgc6i1.png)​
‍

在左侧菜单栏选择「语言」，点击右侧的「中文」，再点击选项：

​![](https://image.peterjxl.com/blog/image-20241107103101-iv29hf6.png)​
‍

拉到下方选择「添加键盘」，然后选择添加微软拼音

​![](https://image.peterjxl.com/blog/image-20241107103740-9w87vam.png)​
‍

单击新添加的微软键盘，然后点击选项：

​![](https://image.peterjxl.com/blog/image-20241107103805-5zkqu4b.png)​
‍

设置中选择「常规」：

​![](https://image.peterjxl.com/blog/image-20241107104004-x9rtvyp.png)​
‍

在拼音设置中选择双拼：

​![](https://image.peterjxl.com/blog/image-20241107104037-g3olgp5.png)​
‍

然后就可以在下方选择方案了：

​![](https://image.peterjxl.com/blog/image-20241107104143-roiu2j1.png)​
‍

## Linux

查了下，有些发行版内置了双拼（例如 Ubuntu 的 IBus），如果没有的话可以安装流行输入法，然后切换为双拼。

推荐用 Rime，可以通过 apt 等方式安装，非常方便。

如果想用大厂输入法，可以试试搜狗，百度等，可以看我另一篇文章，里面写了各大输入法支持的平台：《[十款好用的输入法软件推荐](https://www.peterjxl.com/typewriting/software-recommend/)》。
‍
