# 1100.InputTip：输入法状态提示工具，让你的输入更高效

​![](https://image.peterjxl.com/blog/image-20241012173107-rfh2wn9.png)​

InputTip 是一个输入法状态（中文/英文/大写锁定）提示工具，免费开源，基于 AutoHotKey 开发。

‍<!-- more -->

## 介绍

​![](https://image.peterjxl.com/blog/image-20241008201431-of5gt3u.png)​

官网：[https://inputtip.pages.dev](https://inputtip.pages.dev)

开源在 GitHub：[https://github.com/abgox/InputTip](https://github.com/abgox/InputTip)  和 Gitee：[https://gitee.com/abgox/InputTip](https://gitee.com/abgox/InputTip)

特点：

* 根据输入法状态（中文/英文/大写锁定），显示不同的鼠标样式
* 根据输入法状态（中文/英文/大写锁定），在输入光标附近显示不同的符号
* 有丰富的配置项，高度可定制
* 兼容各种流行输入法（极少数输入法不支持，详情：https://github.com/abgox/InputTip/issues/3）
‍
## 安装

直接点击下载按钮，会得到一个 `InputTip.exe`​ 文件，双击即可运行。

​![](https://image.peterjxl.com/blog/image-20241012174839-5s1cura.png)​
‍

官方推荐做法：新建一个目录，将 `InputTip.exe`​ 放入其中，然后再运行。因为运行 `InputTip.exe`​ 后，会产生不少文件。这样做的话，所有相关的文件或文件夹都在同一个目录中，方便管理。
‍
运行后，就可以看到任务栏有个托盘菜单，右键可以看到不少设置：

​![](https://image.peterjxl.com/blog/image-20241008203101-g4rvqx2.png)​
‍

## 使用

在运行后，就可以看到任何一个输入框的光标旁边，都会有一个小红框：

​![](https://image.peterjxl.com/blog/image-20241008203520-vh077kl.png)​
‍

而如果切换为大写模式，就会变成一个绿色的框：

​![](https://image.peterjxl.com/blog/image-20241008203542-g5rrxfb.png)​
‍

这些可以在设置里改：

​![](https://image.peterjxl.com/blog/image-20241008203634-n15i0kn.png)​

## 常见问题

* 为什么有些界面不显示提示？ 普通用户权限启动该软件在管理员权限的界面中可能无法显示，这是系统决定的，可使用管理员权限启动该软件
* 为什么不在鼠标附近显示？因为打字的时候注意力在光标处而不在鼠标处，鼠标附近显示很好实现，但是没啥用，一般可以在遇到不支持的软件时回退到鼠标附近显示。
* 杀毒软件误报：请使用火绒杀毒，360 等一众的误报太过于离谱无力解决，如遇火绒误报请联系我解决
