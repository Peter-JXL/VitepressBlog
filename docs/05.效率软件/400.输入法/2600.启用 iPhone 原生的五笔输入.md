# 2600.启用 iPhone 原生的五笔输入

聊聊如何在 iOS 中使用原生的五笔输入法

本文虽然介绍的是如何添加五笔键盘，其实其他键盘（双拼，外语键盘、第三方输入法）也是类似的添加方式。

‍<!-- more -->

## 使用原生的理由

虽然之前的文章列了不少第三方的五笔输入法，但其实原生的也不错的：

* 无需安装，运行更稳定，没有各种跳转打开，无需询问完全权限设置，没有第三方输入法的弹出卡顿（例如自动切回原生键盘，我遇到好几次了）。
* 和系统整体风格更统一
* 外接蓝牙键盘时，有时候只能用原生的输入法
* ...


## 启用五笔

首先打开设置，搜索「键盘」，然后点击进入设置页面：

​![](https://image.peterjxl.com/blog/image-20241013113538-k5sl9qk.png)​
‍

PS，也可以在输入法页面里打开「键盘设置」：

​![](https://image.peterjxl.com/blog/image-20241013113532-gzblot2.png)​


然后点击「键盘」，进入键盘管理页面：

​![](https://image.peterjxl.com/blog/image-20241013113459-o571d6o.png)​
‍

点击「添加新键盘」：

​![](https://image.peterjxl.com/blog/image-20241013113524-69xdhbl.png)​
‍

然后选择「简体中文」，再选择「五笔」：

​![](https://image.peterjxl.com/blog/image-20241013113518-oa61tza.png)​


然后就可以切换输入法为五笔了：

​![](https://image.peterjxl.com/blog/image-20241013113507-k4129nf.png)​
‍

## 设置输入法

添加新键盘后，还可以对其进行设置，选择对应的五笔版本

​![](https://image.peterjxl.com/blog/image-20241013113717-5mfwwc8.png)​


## 更多小技巧

拼音检索：有时候会记不起某个字怎么拆，或者怎么写，这时候我们可以直接打拼音，也能打出来，并且还会贴心地告诉你这个字的五笔编码是什么，让你学习怎么拆解

​![](https://image.peterjxl.com/blog/image-20241013160348-6p1z6dx.png)​
‍

此外，之前介绍的输入法技巧，在五笔一般也适用，例如

* 长按打出特殊字符
* 双击空格，快速输入句号
* 自定义短语
* 长按空格键，进入触控版模式，可以快速移动光标
* ...
