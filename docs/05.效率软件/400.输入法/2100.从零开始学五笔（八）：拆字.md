# 从零开始学五笔（八）：拆字

学会字根后，下一步就是学习如何「拆字」了，也就是将一个字拆成几个字根，然后就可以按下对应的按键，开始打字。

‍<!-- more -->

## “单”结构汉字

所谓的“单”结构汉字，就是指构成该汉字的字根只有一个，主要包括：

* 键名汉字：字根图上每个键左上角的黑体字；也是每句口诀的第一个字（X 键除外）
* 字根汉字：除了键名汉字外，本身既是字根也是汉字的字

输入这些汉字时，不用再进行拆分。例如：水，小，力，虫，五，广，八，车，马，九，几，巴，上，早，弓，止...
‍
## “散”结构汉字

所谓的“散”结构汉字，就是构成该字的字根有多个，而且每个字根直接有明显的距离，既不相连也不相交。

例如：明，佣，监，浅，仿，打，伙，咪，务，树，对，字，故，花，只，休
‍
## “连”结构汉字

“连”结构汉字是指由一个单笔画字根与一个基本字根相连而构成的汉字。例如：

自 = 丿 + 目

千 = 丿 + 十
‍
在区分“连”结构汉字时，应注意以下两点：

1. 若单笔画与基本字根有明显的距离，则不属于“连”结构。例如：“少”、“么”、“个”等。
2. 若一个汉字是由一个基本字根与一个点组成，也属于“连”结构汉字。如“义”、“术”、“勺”等。
‍
## “交”结构汉字

“交”结构汉字指由几个字根互相交叉构成的汉字，字根与字根直接相互交叉重叠。

例如：内，果，夫，用，鬼，事，本，丰，中，巾，甩，申


## 汉字的拆分原则

拆分的原则有：

* 书写顺序
* 取大优先
* 兼顾直观
* 能散不连
* 能连不交

下面逐个进行讲解。


### 书写顺序

在书写汉字时，讲究  “先左后右，先上后下，先横后竖，先撇后捺，先内后外，先中间后两边，先进门后关门”的顺序。例如：

* 先左后右，例如「林」
* 先上后下，例如「等」，「戊」（先写上面的点）
* 先横后竖，例如「十」
* 先撇后捺，例如「八」
* 先内后外， 例如「道」
* 先中间后两边，例如「非」
* 先进门后关门 ，例如「国」
‍
拆分字根时，也应该按照书写顺序，拆分为基本字根（顺序很重要）：

|汉字|正确拆分|错误拆分|
| :--: | :------: | :------: |
|伙|亻 + 火|火 + 亻|
|取|耳 + 又|又 + 耳|
|宁|宀  + 丁|丁 + 宀|
|要|西 + 女|女 + 西|
‍
正确拆分后，接下来就是根据顺序来输入字根了。

例如刚刚提到的「伙」就是先按下「亻」字根所在的 w 键，然后是「火」字根所在的 o 键：

​![](https://image.peterjxl.com/blog/image-20241013194951-bj20wx9.png)​


又比如：

* “新”的字根顺序应该是“立、木、斤”，而不能取“木、立、斤”，也不能是“立、斤、木”
* “和”的字根只能取“禾、口”，不能取“口、禾”
‍
### 取大优先

取大优先指的是按书写顺序拆分时，拆出的字根尽量“大”，以保证拆分出的字根数最少。例如：

* “适”可以拆为“丿、古、辶”，还可以拆成“丿、十、 口、辶”。哪一种是正确的呢？根据取大优先的原则，拆出的字根要尽可能大，而后二种拆法中的“十”、“口”两个字根可以合成为一个字根“古”，所以前一种拆法是正确的。
* “除”可以拆分成  “阝、八、禾”  或拆成  “阝、八、丿、木”。前一种是正确的，把  “丿、木”  合成为  “禾”。
* “世”字可以取为“一、凵、乙”  也可以取“廿、乙”，后一种是正确的。

更多示例：

|字|正确拆法|错误拆法|
| :-: | :------: | :----------: |
|交|六 + 〤|亠 + 八 + 〤<br />|
|则|贝 + 刂|冂 + 人 + 刂|
|草|艹 + 早|艹 + 日 + 十|
|莘|艹 + 辛|艹 + 立 + 十|

### 能连不交

当一个字可以视作相连的几个字根，也可视作相交的几个字根时，建议拆成可相连的情况。示例：

|字|正确拆分|错误拆分|
| :--: | :--------: | :---------------: |
|千|丿+ 十|丿 + 一 + \||
|于|一 + 十|二 + \||
|天|一 + 大|二 + 人|
|生|丿 + 龶|每字头 + 土|
‍
### 能散不连

拆分汉字时，如果能拆成“散”结构的字根，就不要拆分成“连”结构的字根。

「主」字按取大优先拆分，会得到「亠」和「土」；但根据能散不连的原则，会得到「丶」和「王」，后一个拆法才是对的。

### 兼顾直观

拆字时，尽量照顾字的直观性，一个笔划不能分割在两个字根中。为了使字根的特征明显易辩，有时要牺牲书写顺序的原则。

例如「国」字，如果按书写顺序，其字根应是“门、王、丶、一”，但这样编码不但有违该字的字源，也不如字根“口”直观易辩。为了直观，「国」字的正确拆法是：“口、王、丶”

注意：该原则比较少见，了解即可。


### 小结

一句话小结：应当保证每次拆出最大的基本字根，在拆出字根的数目相同时，“散”比“连”优先，“连”比“交”优先。
‍
