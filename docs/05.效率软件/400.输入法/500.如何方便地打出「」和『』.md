# 如何方便地打出「」和『』

比起英文中的引号 ‘’和 “”，我更喜欢使用中文直角引号：「」和 『』。

‍<!-- more -->

此外，在港澳台、日本这几个地区中，就经常使用『』和「」：

​![](https://image.peterjxl.com/blog/image-20240902111310-rx0b712.png)​

注意：不同地区的习惯可能有所不同。在汉语中『』、「」分别为双引号和单引号；但在日本，「」是引号，而『』是书名号。

下面来说说如何方便地打出这两个符号。
‍
## 自定义短语

首推的方式是使用[自定义短语](https://www.peterjxl.com/typewriting/typewriting-tips/#自定义短语)，例如我是这样写的：

```
; 排版
kh=2,「」
kh=3,『』
```
‍
## `Win + .`​

​`Win + .`​ 除了可以调出 emoji 面板，也可以调出「符号输入面板」：

​![](https://image.peterjxl.com/blog/image-20240902163000-jmb2l27.png)​
‍

## 修改默认标点符号

基本上各大输入法都自定义标点符号，可以直接把引号设定为「」『』。例如，我使用的手心输入法可以这样设置：

​![](https://image.peterjxl.com/blog/image-20240902112921-7vcodqw.png)​

‍
还有的输入法自带了特殊符号输入模式。例如微软输入法，切换到中文输入后按下 uubg 便可打出特殊的标点符号：

​![](https://image.peterjxl.com/blog/image-20240902160107-zw87e5n.png)​

‍
有些输入法还支持调出符号大全的面板，例如搜狗：

​![](https://image.peterjxl.com/blog/image-20240902160906-wj0la2s.png)​
‍

## 使用 Windows 的 Alt 键

如果不想设置输入法，在 Windows 上可以用以下方法来完成输入：

* 「：摁住「Alt」键 ，小键盘输入「41400」，松开 「Alt」键
* 」：摁住「Alt」键 ，小键盘输入「41401」，松开 「Alt」键
* 『：摁住「Alt」键 ，小键盘输入「41402」，松开 「Alt」键
* 』：摁住「Alt」键 ，小键盘输入「41403」，松开 「Alt」键

具体原理可以看我的上一篇博客 🙃。
‍
## 最后

本文虽然介绍的是如何方便地打出「」和 『』，其实对于其他特殊符号也可以用类似的方法。
