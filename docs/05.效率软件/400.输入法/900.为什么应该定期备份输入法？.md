# 900.为什么应该定期备份输入法？

我们平时使用输入法，不可避免地会积累很多自己的词库、自定义短语和相关设置。如果哪天换新设备了、换电脑/手机了，那么备份的重要性就体现出来了。

‍<!-- more -->

## 为什么要备份？

1. 当你用了一段输入法后，平时经常打的字、短语或人名，输入法都会帮你记住；如果哪天电脑坏了或者换了新电脑，之前经常打的字，新的输入法是不知道的，得重新用了一段时间后，才会顺手。
2. 安装输入法后，一般都会进行很多的设置，例如外观，快捷键等，如果换了电脑或重装了输入法，重新设置一遍很麻烦。

如果有备份，可以在新电脑上导入词库或者设置，让输入法跟之前一样顺手。
‍
## 要备份什么？

其实要备份的东西也不多，就三样：用户词库、自定义短语、设置。

注意，不是所有输入法都有导入/导出功能，所以有些东西没法备份，这没办法🙁

这里以手心输入法为例（其他输入法都差不多），讲解下如何备份。

## 导出后怎么保存？

为了防止如果电脑损害导致备份丢失，建议可以定期导出词库，加密后存放到网盘 / U 盘上。

我个人建议是一个月备份一次，或者半年。

我会给自己定一个月度惯例，每个月定期做一些周期性的事务，其中就包括了输入法的备份。
