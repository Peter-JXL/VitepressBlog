# 提高打字速度的 N 种方法

在互联网时代，打字速度快挺好的

‍<!-- more -->

## 方法论

1. 选定输入法：熟练掌握五笔或双拼后，打字速度会比全拼、简拼快很多，建议学习。
2. 挑选一个键盘：键盘的排列有很多种，我个人偏好 104 键的。而手机输入法我倾向于全键盘。
3. 指法：学会正确的指法，而不是用“二指禅”打字
4. 学会盲打：如果还要看键盘才能打字的话，速度会非常非常慢。因此有必要集中一段时间练习，将所有键位都掌握
5. 多练习：学会盲打后，多练习能很快地提高打字速度。

## 金山打字通

​![](https://image.peterjxl.com/blog/image-20241008105028-fgn655b.png)​

官网：[https://www.51dzt.com/rubik-ssr/51dzt](https://www.51dzt.com/rubik-ssr/51dzt)

首推，非常适合新手，从手的摆放姿势开始学起，再到每个键位的学习，还有盲打练习、速度测试、打字游戏等

> 不过我感觉这个软件有点流氓 😑


## 极速打字网

​![](https://image.peterjxl.com/blog/image-20241008105143-7lryk7b.png)​

官网：[https://www.jsxiaoshi.com](https://www.jsxiaoshi.com)

不仅仅是一个在线练习打字的网站，更包含了社区、排行榜等功能，玩玩还是可以的。


## Type.fun（打字星球）

​![](https://image.peterjxl.com/blog/image-20241008105213-ik9qp15.png)​

官网：[https://www.type.fun](https://www.type.fun)

官方介绍：丰富的盲打课程，科学的打字课程设计，诗词歌赋，经典名著，少儿编程，背单词，小键盘，练习打字同时收获更多，结合跟读轻松学会拼音，百万用户选择的打字练习网站


## EdClub

​![](https://image.peterjxl.com/blog/image-20241008105738-knlgnqt.png)​

官网：[https://www.typingclub.com](https://www.typingclub.com)

在线网站（支持多种语言），用闯关的方式来练习打字，还是蛮有趣的。

## 最后

盲打没什么技巧，唯手熟尔。先集中一段时间掌握盲打，后续打字坚持使用盲打就相当于的练习了，只会越来越熟练。
