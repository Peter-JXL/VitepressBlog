# 从零开始学双拼（五）：练习

来讲讲双拼学习最重要的部分：练习

‍<!-- more -->

## 方法论

1. 先掌握双拼方案，至少混个脸熟。这一步请参考上一篇文章。
2. 开始练习。可以用金山打字通，可以用一些在线练习网站，如果有码字的习惯，也可以直接开始写文章
3. 遇到不会的，再去看双拼方案图，记忆起来会更牢固
4. 先保证正确率，再提高速度，避免养成错误的肌肉记忆
5. 把手机输入法也切换成双拼，强迫自己使用，能更快记住和适应键位

强烈建议专门抽出一两天来练习，一两天其实完全足够我们记住所有键位了，然后就可以放下全拼，开始使用双拼了。

如果一两天的时间都没有，那至少希望你能抽出两三个小时记忆键位，先掌握双拼方案，达到能用的程度，然后就开始用双拼，一开始慢一点没关系的，开始用双拼后，后续的每次打字都能让你更熟练。


## 关于方案


为了方便看双拼方案图，我们可以将双拼方案打印下来贴墙上，甚至打印多几张，放在家里不同的位置都可以。

或保存到相册，可以在坐车、等人时，默默背诵。

还可以设为壁纸，忘记键位的时候就看一眼。例如在 Windows 下，可以用快捷键 `Win + 逗号键` 快速预览桌面。

更有甚者，定做了一张键位卡（类似银行卡），其实就是打印一张很小的纸，然后贴到卡上，这样非常方便随身携带和背诵。

## 做好记录

可以考虑记录打字速度：

* 先记录自己的拼音打字速度
* 然后练习双拼，分别记录一个月后，三个月后的打字速度

通过比较，才知道双拼是不是值得学习的。
