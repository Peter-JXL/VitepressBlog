# 200.十款好用的输入法推荐

书接上回，在提高我们的打字速度、优化打字体验之前，得先挑一款合适的输入法。

本文就来简单介绍一些好用的、高颜值的输入法。
‍<!-- more -->
‍

## 搜狗输入法

​![](https://image.peterjxl.com/blog/image-20241004212933-k1v1a9z.png)​

官网：[shurufa.sogou.com](https://shurufa.sogou.com)

搜狗输入法可以说是老牌输入法了，起步很早。从使用率来说，搜狗已经稳坐第一好多年了，说是王者也不为过。

特点：

* 输入方式：全拼、简拼、五笔（需下载[搜狗五笔输入法](https://wubi.sogou.com)）、语音输入、手写输入、OCR
* 支持的平台：Windows，Mac，Linux，Android，iOS
* 强大的云词库，能方便地打出各种网络流行语
* 开始集成 AI，打字更智能、准确
* 支持企业版，为 B 端行业定制多场景输入方案（不过个人应该用不上🤔）
* 强大的皮肤系统：支持换皮肤，自制皮肤，还有一个皮肤集市。我当时选择它就是看上了那些炫酷的皮肤 🤣
* ...
* 缺点：臃肿，广告多。为此网上还有不少“魔改版”，号称自己去除了广告。我的建议是小心为上，用官方的输入法 + 广告拦截器（例如火绒就自带拦截弹窗功能）
‍
此外搜狗还推出了[医生版](https://pinyin.sogou.com/yisheng/)，相当于官方推出的去广告纯净版，最后一次更新在 2018 年 9 月。

有丰富的关于医学的词库，但是没有账户登录功能，无法云同步。如果你喜欢搜狗输入法，但不想要广告的话，可以考虑使用这一款。

​![](https://image.peterjxl.com/blog/image-20241005165146-g3yoj4a.png)​
‍‍

## 百度输入法

​![](https://image.peterjxl.com/blog/image-20241004215119-3f2h172.png)​

官网：[shurufa.baidu.com](https://shurufa.baidu.com)

也是很老牌的输入法了，在很多方面都和搜狗输入法类似... 只能说，这很百度。

特点：

* 输入方式：全拼、简拼、双拼、五笔、语音输入、手写输入
* 支持的平台：Windows，Mac，Linux、Android，iOS
* 集成 AI：百度的文心一言还是比较出名的
* 强大的皮肤系统：和搜狗类似，可以自制皮肤，也有一个皮肤超市
* 支持企业版，为 B 端行业定制多场景输入方案
* ...
‍
## 微信输入法

​![](https://image.peterjxl.com/blog/image-20241004214155-2y1aqqw.png)​

官网：[z.weixin.qq.com](https://z.weixin.qq.com)

微信推出的输入法，界面很不错，功能还算齐全，值得一试。

特点：

* 输入方式：全拼、简拼、双拼。手机版可以用笔画、五笔、语音、手写
* 支持的平台：Windows，Mac，Android，iOS
* 支持跨设备：在电脑复制图片、文字后，在手机上可立即粘贴
* 其他特点：在多端同步常用语（这个很不错），集成 AI，智能翻译，无广告
* 界面：只有浅色、深色模式，没有换肤系统
* ...

## QQ 输入法

​![](https://image.peterjxl.com/blog/image-20241005103757-l9zwpkt.png)​

官网：[qq.pinyin.cn](https://qq.pinyin.cn)

特点：

* 输入方式：全拼、简拼、双拼
* 支持的平台：Windows，Android
* 支持换肤，自制皮肤，有皮肤集市
* ...

个人感觉，和之前几款大厂的输入法比起来，QQ 输入法要逊色不少，支持的平台少，更新频率低（上次更新是 2020 年了）
‍
## Google 输入法

​![](https://image.peterjxl.com/blog/image-20241005153306-nuhzjqz.png)​

也是一款很老牌的输入法了，大厂出品，而且较早地搞了全球各种文字的输入系统。

官网：[https://www.google.com/intl/zh-CN/inputtools](https://www.google.com/intl/zh-CN/inputtools)

特点：

* 输入方式：全拼、简拼、双拼
* 支持的平台：Windows、Android
* 功能强、体积小、干净美观、无广告、支持多国语言、翻译、注重隐私
* 词典功能：能牢牢记住输入过的短语和人名，并且能多设备间同步、导入导出等
* ...

除了「语音转文字」功能需要科学上网，其他一切正常。

虽然很久没更新了，但用的人还是不少，在应用商店里评分很高。

​![](https://image.peterjxl.com/blog/image-20241004205940-sc9egaz.png)​

微软输入法，是每个 Windows 都预装了的，经过多次更新，不管是界面还是功能都已经很完善了，非常值得一试。

## RIME 输入法

​![](https://image.peterjxl.com/blog/image-20241005100752-1qx585z.png)​

官网：[rime.im](https://rime.im)

一个开源的、安全的、可以高度定制化的、专业的输入法，门槛较高。

特点：

* 输入方式：全拼、简拼、双拼、五笔
* 支持的平台：Windows、Mac、Linux、Android
* 完全开源，不用担心数据安全问题

市面上的成品输入法，你所能定制的自由，被框限在「设置」标签里。

有别于此，RIME 突出的就是自由的特性。你需要什么功能，需要什么输入方案，输入细节上有什么与常人不同的习惯和需求，它几乎都能满足。

但也正是因为 RIME 的高度自由，引入的高度个性定制化，使得它有着很高的准入门槛。定制困难，劝退了绝大多数人。

即使 RIME 有着种种不足和缺点，但不可否认，它，就是目前唯一能实现你几乎所有输入个性化定制需求的最好选择。

‍
还有的网友这样[点评](https://www.zhihu.com/question/448201288/answer/2578087280)：

> Rime 就像鱼腥草。
>
> 鱼腥草做起来超麻烦，爱吃的人爱得不得了，不爱吃的人完全无法接受。
>
> Rime 虽然有跨平台（官方支持 Win/Mac/Linux，第三方支持 iOS/Android）的优势，但是配置起来超麻烦，会写配置文件的人用起来如鱼得水，你可以打造完全属于你的输入法，但是小白用户用起来如坐针毡，装个五笔输入包都很麻烦（要用命令行从 GitHub 下载，难倒好多小白）。
>
> 所以我把 Rime 放在最后。

## 用什么输入法？

其实，大部分输入法的功能都差不多，该有的功能都有，先理清自己的需求，挑一个自己喜欢的就差不多了。

就比如我，我不需要太多花里胡哨的功能，只要能安安静静地打字就行。因此，一直用的手心输入法，无广告。其他一些大厂的输入法，有时候挺烦人的，广告多，我要用都得先安装一些广告拦截工具。

如果你懒得折腾：推荐搜狗、百度，不过注意安装一些广告屏蔽软件（例如杀毒软件火绒就有弹窗拦截功能）

如果你偏爱语音输入：推荐讯飞输入法，它家的识别功能很强，最近还号称用 AI 加强了。

如果你想自己定制：推荐 RIME，可玩性很高。

至于微信输入法，我感觉还不太成熟，待后续观察吧。

而微软输入法，虽然经过了多次更新，但感觉和目前市面上的输入法，还有一定的差距。
