# 2300.从零开始学五笔（十）：练习

前面我们花了那么多篇幅去学习五笔的规则，并讲解了如何用五笔输入各种汉字，接下来就是练习了。
‍<!-- more -->
‍

## 是否要学习五笔？

通过之前的文章，相信你对五笔已经有基本的认知了。

那么，到底是否需要学习五笔呢？

我们可以看看这篇文章：[二〇二二年，来试试双拼输入法吧 - 少数派](https://sspai.com/post/72622)，里面讲了不少关于五笔的缺点。

如果你还是想试着使用五笔，那就继续往下看吧。
‍
## 怎么练习？

我强烈建议专门抽出几天来练习五笔，达到能打大部分字的程度，然后我们就可以放下拼音，开始在日常中使用五笔了。而随着使用的时间增长，打字也会越来越熟练。
‍
基本方法论：

1. 熟悉规则。之前的几篇文章，建议都好好阅读一次。日后的练习和使用过程中，如果遇到问题要积极复习。
2. 熟悉字根口诀（大概得 2 ~ 3 天），这个主要是靠自己背
3. 熟悉拆字原则，输入规则，这个主要是自己看，然后在后续练习中运用
4. 练习字根（建议用专门的软件练习）
5. 练习简码的输入，一、二级简码必须做到条件反射。汉字是单字使用非常集中的文字，常用 500 字的覆盖率在 78%，常用 1000 字的覆盖率在 92%，常用 1500 字的覆盖率在 97%，五笔想要提速一定得想办法集中练常用字。
6. 练习文章的输入


通过上述练习（大概一周），我们就可以放下拼音，开始使用五笔了。使用过程中建议：

1. 日常要坚持使用五笔，不要混用拼音，这个阶段要慢，要准确不要速度。不知不觉输入速度就会上来的。
2. 一般来说，两个月就可以将打字速度提到 60 字/分钟，一年内能提高到 120+。
3. 如果想尽快提高打字速度，可以参考我之前的文章《提高打字速度的 N 种方法》，先集中一段时间练习几天。也可以用几天时间水水群（逃
4. 每日记录自己卡壳和不会输入的字，然后针对性训练（例如每个字输入 20 遍以上）。
‍
在我学五笔时，我将电脑和手机壁纸都设为了五笔的键位图，甚至打印了出来，没事的时候就看看它，遇到不会的字也会去看，尝试自己拆出来。如果还是不行，就专门去查询这个字的笔画，然后记下来。抱着这种态度去学，一定能学会的。

例如这个网站会列出字的五笔拆法，五笔编码，然后我会单独记下来怎么打：

​![](https://image.peterjxl.com/blog/image-20241024163133-sp3zcdk.png)​
‍

建议做好打字速度的记录：

* 可以先记录自己的拼音打字速度
* 然后练习五笔，几个月后记录打字速度
* 一年后再记录自己的打字速度（选做）

通过比较，才知道五笔是不是值得学习的。
