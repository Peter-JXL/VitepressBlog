# 1300.Input-Source-Pro：自动切换输入法并提示状态

​![](https://image.peterjxl.com/blog/image-20241009112146-zyfhcfx.png)​

Input Source Pro 是一款 macOS 上的输入法辅助工具，它可以根据不同应用、不同网站来自动切换输入法，并可以在鼠标周围显示当前输入法状态。

我个人认为，macOS 对于输入法状态的保存不像 Windows 那样智能，因此这样的软件还是挺有用的。

‍<!-- more -->

## 介绍

官网：[https://inputsource.pro](https://inputsource.pro)

主要功能：自动切换输入法，显示输入法状态。

下面简单演示下。
‍
## 显示输入法状态

可以在切换输入法、应用或长按鼠标左键时，触发提示（显示输入法状态）。

效果：在便签里显示目前的状态是中文输入：

​![](https://image.peterjxl.com/blog/image-20241014201405-d7f6c4c.png)​
‍

在 IDE 里则显示目前是英文输入：

​![](https://image.peterjxl.com/blog/image-20241014201459-add7od5.png)
‍

长按鼠标左键直接显示：

​![](https://image.peterjxl.com/blog/image-20241009112024-io20x0h.png)​
‍

具体可以在软件设置里查看：

​![](https://image.peterjxl.com/blog/image-20241009111914-deu4a18.png)​
‍

## 自动切换输入法

可以在不同应用之间自动切换输入法，例如在终端用英文，而在浏览器自动切换为中文。

甚至在不同网站之间自动切换输入法，例如在 Notion 是中文输入：

​![](https://image.peterjxl.com/blog/image-20241014200720-h5kw81h.png)​

‍
在 GitHub 则切换为英文输入：

​![](https://image.peterjxl.com/blog/image-20241014200736-l7lptfc.png)​
‍

这个功能目前支持 Safari、Chrome 与 Edge 三种浏览器，并且拥有三种策略：DOMAIN，DOMAIN-SUFFIX，URL-REGEX。

> 从策略名字来看，应该是通过域名、前缀、正则等方式来切换。
‍
当配置妥当，积累到一定程度之后，基本能实现再也不用自己手动切换输入法了。
‍
## 最后

总的来说是一款不错的软件，有需要的同学可以试试（目前在测试期间，免费使用）。

macOS 下类似的工具还有：KeyboardHolder，SwitchKey，就不再单独介绍了。
