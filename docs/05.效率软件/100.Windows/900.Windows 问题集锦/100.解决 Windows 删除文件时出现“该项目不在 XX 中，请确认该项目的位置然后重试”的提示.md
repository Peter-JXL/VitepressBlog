# 解决 Windows 删除文件时出现“该项目不在 XX 中，请确认该项目的位置然后重试”的提示

准备在删除一个无用的文件夹，却提示“找不到该项目  该项目不在 G:\中，请确认该项目的位置，然后重试”：

![](https://image.peterjxl.com/blog/2024-08-30_18-07-12-20240830205421-4arjvxr.png)​

‍<!-- more -->

解决方法：

1. 新建一个文本文件，例如 `1.txt`​
2. 在里面输入如下内容：

    ```bat
    DEL /F /A /Q \\?\%1
    RD /S /Q \\?\%1
    ```
3. 将该文件的扩展名从 `.txt`​ 改为 `.bat`​，例如 `1.txt`​ ➡️ `1.bat`​
4. 将删除失败的文件（夹）拖到该 bat 文件上即可
‍
备用方案：使用 winrar，将该文件夹添加到压缩文件并删除原文件。
