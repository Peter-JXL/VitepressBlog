# 查询 B 站注册时间

有时候想看看自己玩 B 站多少年了，想知道自己什么时候注册的。

此外，据说注销 B 站账户的话也得提供详细注册日期。

‍<!-- more -->

## 通过创作中心查看

登录网页版 B 站，点击右上角的创作中心，然后就能看到在 B 站多少天了：

![](https://image.peterjxl.com/blog/image-20240905160733-zo95ngn.png)​


然后可以找一些在线工具进行计算，例如 [https://calcdate.buyaocha.com](https://calcdate.buyaocha.com)：

![](https://image.peterjxl.com/blog/image-20240905160724-pba82xz.png)​


或者问下 AI：

![](https://image.peterjxl.com/blog/image-20240905154634-5hdn0yb.png)​

> 注意：要加上一天才更准确。例如你今天注册 B 站的话，那么 1 天前是昨天，是没有注册的，因此得加上一天。


手机端同理，点击左上角头像，打开创作中心中的创作首页，即可在屏幕正中的滚动栏中看到注册天数了。

注意：得先发布了一个视频后，才能在手机端 App 看到创作中心。

## 通过 API 接口

首先，在网页上登录 B 站，然后打开这个网址：[https://member.bilibili.com/x2/creative/h5/calendar/event?ts = 0](https://member.bilibili.com/x2/creative/h5/calendar/event?ts=0)

会看到一大堆的字符串，我们不用看这些，直接 Ctrl + F 搜索 `jointime`​，就能看到一串数字：

![](https://image.peterjxl.com/blog/image-20240905150557-d19659v.png)​
‍

这个数字就是时间戳，指的是从 1970 年 1 月 1 日开始所经过的秒数，用于表示一个时间点。

然而，这种格式对于人类阅读并不友好，因此需要转换成可读的日期和时间格式。

例如，我们可以使用一些在线工具帮助我们转换，例如 [https://tool.lu/timestamp](https://tool.lu/timestamp)：

![](https://image.peterjxl.com/blog/image-20240905150851-np7k4vi.png)​
‍

同样的，可以让 AI 帮你转换下时间戳，这里就不演示了。
‍
也可以用编程语言，例如 Python：

```Python
> python
>>> import time
>>> time.localtime(1477968045)
time.struct_time(tm_year=2016, tm_mon=11, tm_mday=1, tm_hour=10, tm_min=40, tm_sec=45, tm_wday=1, tm_yday=306, tm_isdst=0)
```
‍
JS：

```Python
new Date(1477968045000).toLocaleString();
'2016/11/1 10:40:45'
```

> 由于该 JS 对象传入的是参数是毫秒单位的，因此得加上 3 个 0 变成秒。
