# 批量下载 B  站动态页图片

在逛 B 站的时候，总能看到不少 UP 会发很多图片，此时一个一个保存非常麻烦，而且文件名都是随机的字符串，还得手工重命名。

为此，特地搜索了下有没相关的浏览器插件或油猴脚本，还真给我找到一个。

<!-- more -->

脚本地址：[https://greasyfork.org/zh-CN/scripts/421885](https://greasyfork.org/zh-CN/scripts/421885)

![](https://image.peterjxl.com/blog/image-20240913115350-p6lezrx.png)​


安装后，重新刷新网页，即可在“BiliBili 动态”[时间线页面](https://space.bilibili.com/10085983/dynamic) 上批量下载图片：

![](https://image.peterjxl.com/blog/image-20240913173655-kp6mmz3.png)​


也可以在 [动态详情页](https://www.bilibili.com/opus/965535242583539713) 里下载：

![](https://image.peterjxl.com/blog/image-20240921101453-vk1sycd.png)​


还可以在设置里自定义下载后的文件名：

![](https://image.peterjxl.com/blog/image-20240913173940-wt3poj6.png)​

也能在时间线和播放页面下载视频，但只能下载 720p 清晰度的，具体请看官方的 [说明文档](https://github.com/owendswang/Download-Pictures-from-Bilibili-Timeline/blob/main/README.zh.md) 和设置页面。
