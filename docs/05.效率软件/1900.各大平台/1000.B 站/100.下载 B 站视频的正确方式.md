# 100.下载 B 站视频的正确方式

使用 B 站也快十年了。作为一只数据仓鼠，我热衷于下载那些优秀的视频到本地，一方面是不用担心该视频被和谐的问题，另一方面是本地观看不用担心网速的问题，还能方便地处理视频（例如提取音频、二次剪辑等）。

> 对于网上的资源，「落袋为安」这四个字都快成我的信仰了 🙃

在此期间，也接触了不少能下载 B 站视频的工具，这次就来一次性分享下。

‍<!-- more -->

## B 站自带的视频缓存功能

这个是官方的功能，可以直接在官方客户端下载视频。

可以选清晰度，速度和体验还是不错的，也是最直接和方便的。

​![](https://image.peterjxl.com/blog/image-20240905175046-lr9qemp.png)​


缺点：部分版权受限的视频无法下载，此外当下载的视频多了起来后不好整理，且手机/平板的空间是有限的。

### 唧唧 Down

​![](https://image.peterjxl.com/blog/image-20240905161327-08dlgx3.png)​

首推的就是「唧唧」了，能下载 99% bilibili 的视频，还能下载弹幕、视频封面、音频等等。

官网：[http://client.jijidown.com](http://client.jijidown.com)

特点：

* 跨平台：支持 Windows、macOS、Linux
* 分为免费版（1.0）和付费版（2.0），免费版基本够用了（我用了七八年了）
* 支持历史记录功能，视频清晰度选择（得登录 B 站），支持批量下载多 p 的视频
* 2.0 的下载速度很快（1.0 是有点限速的），能下载的视频清晰度更高，还支持批量下载 up 主投稿，up 主合集和列表，个人收藏夹...
* 以前我还下载过番剧，现在不行了 🙁，现在我都是下载番剧的弹幕，然后和其他渠道下载的番剧搭配食用

更多功能请参考[官网文档](http://client.jijidown.com/about/)。

​![](https://image.peterjxl.com/blog/image-20240905193719-2aqoyeo.png)​

特点：

* 适用于 Chrome 和 Edge 浏览器，理论上其它使用 Chromium 内核的浏览器也可以使用（如 QQ，搜狗）

## 命令行工具：you-get

​![](https://image.peterjxl.com/blog/image-20240906162013-ti8i6tq.png)​

You-Get 是一个使用 Python 开发的小型命令行实用程序，可以通过一行命令直接从 Web 下载媒体内容（视频，音频，图像）等，不用任何配置。

官网：[you-get.org](https://you-get.org)，需魔法上网

特点：

* 安装简单，使用简单
* 开源在 GitHub：[https://github.com/soimort/you-get](https://github.com/soimort/you-get)，目前已有 50k 的 star
* 支持的平台众多：

  * 音乐类：网易云音乐、酷狗音乐、酷我音乐...
  * 视频类：AcFun、bilibili、优酷、爱奇艺、芒果 TV、央视网、腾讯视频、抖音...
  * 社交类：豆瓣、微博、QQ...
  * 更多支持站点参考 GitHub 文档
* 支持批量下载、指定清晰度与格式、指定保存目录、支持使用 Cookie 下载会员视频
* 更新稳定，近十年来都有维护

中文文档地址：[https://github.com/soimort/you-get/wiki/中文说明](https://github.com/soimort/you-get/wiki/中文说明)
