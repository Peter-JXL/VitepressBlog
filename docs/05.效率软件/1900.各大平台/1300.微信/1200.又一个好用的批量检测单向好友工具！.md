# 又一个好用的批量检测单向好友工具！

2025-10-15 补充：该项目已失效，项目主页写了：请使用新项目 [WeFriends](https://github.com/StrayMeteor3337/WeFriends)

<!-- more -->

微信的“删除好友”功能一直被诟病：无法查看单向删除好友。

之前介绍过[「李跳跳·真实好友」](/Wechat/litiaotiao-real-friend/)，通过〝假转账法〞判断是否被好友删除，缺点是检测时间较长，而且后续文章被删除了🙃

最近又发现一个很火的开源项目：[WechatRealFriends](https://github.com/StrayMeteor3337/WechatRealFriends)，也能批量检测，看看有无朋友删掉或者拉黑你，速度很快，结果准确，安利一波。

‍<!-- more -->

## 项目简介

功能简介：快速检测你的微信有没有单向好友，并自动将其添加到标签中 (之后可在电脑端一键清除），理论上可检测上万好友

开源地址：https://github.com/StrayMeteor3337/WechatRealFriends

短短两周，就获得了 4k Star！

![star-history-2025330](https://image.peterjxl.com/blog/star-history-2025330-20250330151837-xcegr51.png)
‍

## 使用指南

在项目发布页下载压缩包：https://github.com/StrayMeteor3337/WechatRealFriends/releases/

![](https://image.peterjxl.com/blog/image-20250330124306-ry2kvxf.png)
‍

为了方便读者，我也转了一份到我的网盘里，读者可以在公众号回复「资料」获取。

解压压缩包，建议先阅读下「使用前必读.txt」，然后双击 exe 文件：

![](https://image.peterjxl.com/blog/image-20250330124445-9fwwap3.png)
‍

然后会弹出一个黑窗口（请勿关闭），并且会弹出浏览器页面（网页地址：http://localhost:8001）：

![](https://image.peterjxl.com/blog/image-20250330124951-okh8bwp.png)

补充说明：

* 若浏览器没反应，可以手动输入网址
* 检测过程不会打扰到好友，可以放心用
* 扫码登录时，要确保在同一网络环境，否则可能无法登录
* 由于软件原理是利用 iPad 的协议，所以授权登录时，显示的设备类型是 iPad，这是正常的
‍
扫码登录后，就会自动开始检测了，速度非常快（一两分钟）：

![](https://image.peterjxl.com/blog/image-20250330125154-wvgwrbf.png)
‍

检测完后，会提示是否给异常好友添加标签，建议添加，否则检测结果会丢失：

![](https://image.peterjxl.com/blog/image-20250330125219-kgv99d8.png)
‍

然后你可以打开通讯录，查看该标签下的好友都有谁，做进一步处理：

![](https://image.peterjxl.com/blog/image-20250330125519-rdnmy4p.png)
‍

## 注意事项

此类项目一般都使用微信协议制作，有可能会触发微信风险检测，不建议经常使用。

我一般是半年/一年检测一次，也基本够用了。
‍
## 最后

比起「李跳跳·真实好友」动辄一两个小时的检测时间，这个项目的检测速度太快了，强烈推荐。

我比较担心的是微信官方会更新相关协议，导致软件失效，且用且珍惜。

如果失效了，再用「李跳跳·真实好友」也不迟，这种模拟人工操作的一般不会被封。
