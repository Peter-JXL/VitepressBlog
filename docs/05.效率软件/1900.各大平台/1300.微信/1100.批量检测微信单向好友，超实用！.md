# 批量删除微信单向好友，超实用！

​![](https://image.peterjxl.com/blog/image-20241113171901-4dsiclk.png)​

随着时间推移，大家免不了加好友。而微信的“删除好友”功能一直被广大网友诟病：无法查看单向删除好友。

> 其实最近（2024-10-23）微信上线了一项新功能，当微信好友量到达 1 万上限后可查单删好友（安卓，iOS 均可）。但对于大多数人来说，门槛太高了吧😅。

为此，开发者“李跳跳”开发了一款 APP「真实好友」，可以检测所有单向好友。刚好最近更新了 5.0 版本，支持安卓微信 8.0.53 及附近的版本，来试试看吧！

## 真的需要删除吗？

在动手之前，我们不妨想想，我们真的需要删除好友吗？

1. 首先，大部分人都是萍水相逢，删不删无所谓的。如果微信好友量达到了上限，再删也不迟。

至于我，纯粹是好奇，才动手试试😂

## 原理

微信有个机制，如果你被好友删了，当你识图给其转账的时候，会失败：

1. 在转账页面会显示实名信息（在微信名后面）
2. 转账前会提示：「你不是收款方好友...」

​![](https://image.peterjxl.com/blog/image-20241113171306-ub95907.png)​
‍

需要强调一下，是通过对话页面与好友进行转账，不是扫描二维码收付款。

「真实好友」通过自动化，尝试给每个好友转账（但不会真的付钱），然后记录结果，给单向好友添加备注。

‍除了这招，还有一些简单粗暴的方法：

1. 逢年过节，群发一些祝福即可。缺点：多多少少会打扰他人，慎用。
2. 拉群聊，如果对方已经删除了你，系统会提示“你无法邀请未添加你为好友的用户进入群聊”。拉群后只要不说话，就不会打扰他人。缺点：一次只能拉几个人。

## 使用教程

先下载 APP。注意只能用在安卓手机上，没有的同学可以试试电脑上装一个安卓模拟器。

使用起来也很简单，分三步就行：

1. 打开 APP，开启无障碍权限
2. 开始检测。可能会比较久，500 个好友大概半小时，但至少不用自己动手了 🙃
3. 检测结束后，会在异常好友的昵称后面加个备注
4. 检测完后最好还是自己也检查下，毕竟这个是模拟转账，并不一定百分百准确
5. 通过微信的好友管理功能，批量删除单向好友
‍
开始检测后，会自动帮你自动打开微信，自动进行转账检测（但不会付款，会取消的，而且没要求你输入付款密码）。

如果检测过程中发生了中断（例如有电话、没电关机了等情况），重新开始时会从中断位置继续。

## 企业微信

如果你平时有用企业微信，其实已经自带该功能了：

​![](https://image.peterjxl.com/blog/image-20241115113905-est21dh.png)​

入口：通讯录--我的客户--微信客户--单向微信客户，就可以进行筛选了。

这两个功能对于企业微信是非常实用的，因为企业微信经常会群发广告很容易被客户单删，时间久了，单删好友也会越来越多。

要知道，企业微信里好友位可是要付费的，一个企业微信号最多可以添加 2000 个免费好友，超过的则要付费 0.1 元一年一个。
