# 1 秒预览各类文件内容，让 Windows 更加丝滑

![](https://image.peterjxl.com/blog/image-20250620105953-5oros28.png)

<!-- more -->

## 前言

一般情况下，在 Windows 上想要确定某个文件的内容，都得先打开它。

当文件过多时，逐个打开 — 查看 — 关闭，其实是一件很麻烦的事情。

如果是文本文件、图片等还好说，如果是 Word、Excel 甚至要用 Photoshop 打开的 psd 文件就很麻烦了。

只是查看文件，也要忍受软件慢悠悠的加载速度，就太低效了。

因此，本文就介绍一下系统自带的预览功能，以及一些好用的预览软件，帮你把时间节省出来。

‍

## 系统自带功能

实际上，Win7 及以上的系统就自带了一个名为「预览窗格」的功能，可以在不打开文件的情况下，快速预览 Word／PPT／Excel／PDF／TXT／图片等文件的内容。

启用方法（以 Win10 为例）：点击最上方的「查看」，再点击「预览窗格」

![](https://image.peterjxl.com/blog/image-20250619113420-slqfagn.png)

PS：我更推荐用快捷键 Alt + P 来打开/关闭预览窗格。

‍

然后，文件夹右侧就会显示预览窗格，当用鼠标选中文件时，预览窗格就会显示文件的内容了

![](https://image.peterjxl.com/blog/image-20250619120238-5do6ymq.png)

‍

还可以用鼠标拖动边界线，调整预览窗格大小：

![preview-windows-size](https://image.peterjxl.com/blog/preview-windows-size-20250619153639-uw1v9be.gif)

‍

进阶使用：

- 注意，预览窗格只能预览，不能编辑
- 支持预览 GIF 的第一帧
- 支持预览 Word，Excel，还可以在窗格内复制、全选内容
- 支持预览 PPT，还可以在窗格内用鼠标或滚轮翻页，右键可以复制幻灯片，能播放幻灯片中插入的视频、音频
- 支持预览 PDF，但不能选中内容，只能翻页。网上查了下，貌似要安装 PDF 软件才能预览，例如 Adobe Acrobat Reader、SumatraPDF 等。
- 支持预览 HTML 文件（但是图片不能正常显示）
- 支持预览音频和视频的封面（不能播放）
- 个人测试：预览图片、音频、视频时，双击窗格则相当于双击该文件，会用默认程序打开它。
- 要切换文件预览的话，除了用鼠标选中文件，也可以用方向键 ↑ ↓ ← → 切换文件

‍

小结：

- 优点：自带功能，无需额外安装软件
- 缺点：挺占用屏幕空间的，部分文件格式不支持预览（例如音视频，压缩包，Markdown，电子书，PSD...）

‍

总的来说，勉强够用，但不够好，不够专业。

预览窗格一般都挺小，看起来不方便；而且会占用屏幕空间，频繁打开 / 关闭窗格也挺麻烦。

---

如果你用 Mac 苹果电脑，则按下空格就会弹出一个新的窗口预览文件，再按空格则关闭，还支持编辑等，是 macOS 原生神级应用。

由于这个功能实在太好用，不少开发者做了类似的工具，让 Windows 也拥有了这个功能！

接下来会着重介绍下免费开源的 QuickLook。

‍

### 日常使用

安装完后，在文件上按下空格键，就会打开一个预览窗口：

![](https://image.peterjxl.com/blog/image-20250619170602-gu4qlif.png)

‍

再次按下空格，就会关闭预览窗口（也可按 Esc 键关闭）。

使用技巧：

- 在预览过程中，按下回车键，就会用默认程序打开该文件
- 预览窗口上的按钮，从左到右依次是：回到顶部、钉住、用默认程序打开、用其他程序打开、分享、最大化、关闭
- 可以拖动边界线来调整预览窗口大小
- 可以在预览窗口内复制内容，上下滚动
- Ctrl + 鼠标滚轮能够缩放图片（或文档）
- 在预览过程中，可以用方向键 ↑ ↓ ← → 切换到其他文件
- 支持从 Everything 软件上按空格直接预览文件的，一个非常不错的功能联动
- 而且在预览音频、视频时，能调整进度条和音量（比起 Windows 自带的那个强太多了）

![](https://image.peterjxl.com/blog/image-20250619172153-vmgrjv5.png)

‍

还能预览文件夹（显示文件夹大小和数量）

![](https://image.peterjxl.com/blog/image-20250619174144-70rnd6c.png)

‍

### 安装插件

QuickLook 支持的格式众多：

- 图片格式：png、apng、jpg、gif、psd、ai..
- 压缩文件：zip、rar、tar.gz、7z...
- 办公文件：doc、xls、ppt、csv、pdf...
- 音视频：mp4、mkv、m2ts、ogg、mp3、m4a...
- 电子邮件：eml、msg...
- 各种文本文件：txt、json、css、xml、md、markdown、htm、html 等，大部分代码文件也支持

‍

但，QuickLook 还支持插件功能，任何人可以写代码，满足其预览特殊格式的需求。

例如，可以预览电子书、apk 安装包、种子文件等，甚至预览文件夹的内容。

QuickLook 项目收集了不少[网友写的插件](https://github.com/QL-Win/QuickLook/wiki/Available-Plugins)，你可以看看有没自己想要的：

![](https://image.peterjxl.com/blog/image-20250619165913-8vgqjud.png)

‍

插件的安装方式也不难：

1. 通过以上列表的 Link 链接，下载到.qlplugin 格式的文件
2. 用 QuickLook 的空格预览来打开插件
3. 然后在打开的窗口点击即可安装
4. 安装好后需重启 QuickLook 使插件生效

‍

例如，下载好文件夹预览的增强插件，按空格预览，点击安装：

![](https://image.peterjxl.com/blog/image-20250619173050-lpcw0i1.png)

‍

然后就能看到文件列表了：

![](https://image.peterjxl.com/blog/image-20250619174552-vmwsmz1.png)

‍

又比如，安装好 apk 预览支持的插件之后，可预览 apk 文件的具体版本号、图标和支持的 CPU 类型，省的打开编程工具查看：

![](https://image.peterjxl.com/blog/image-20250619173338-dum99ps.png)

‍

我还安装了 epub 电子书预览支持的插件，真的帮我省了不少时间。

我也下载了一些插件，分享到网盘上，有需自取。

## 平替软件

类似 QuickLook 的软件有不少：

- [Seer](https://1218.io/seer.html)：收费，但功能更强，速度更快一点，稳定性更强
- WinQuickLook：要求 Win10 1903 以上，更轻量级一点
- ......

使用方法都是类似的，按下空格开启 / 关闭预览窗口，。也支持插件，这里就不多介绍了。

‍

## 其他预览小工具

还有些工具支持右键预览，例如预览压缩包内容的 Bandizip：

![](https://image.peterjxl.com/blog/image-20250620102051-h1eqirb.png)

‍

如果你用 Photoshop，平时会接触到 .psd、.hdr 文件，还可以试试 [SageThumbs](https://www.cherubicsoft.com/en/projects/sagethumbs)，它可以直接让这些文件显示缩略图，而不用打开 PS 查看：

![](https://image.peterjxl.com/blog/image-20250620102708-2b6tggy.png)

‍

这样，安装后甚至不用预览软件，直接用 Ctrl + 鼠标滚轮 ↑ 即可放大文件，让缩略图更大一点。

‍

## 最佳实践

我目前预览文件，主要用的是：

- QTTabBar：预览“小”文件，例如 txt 文本文件，图片
- QuickLook：预览“大”文件，例如音视频，电子书，Office 文件

Windows 自带的预览窗格，几乎不用了，太麻烦。
