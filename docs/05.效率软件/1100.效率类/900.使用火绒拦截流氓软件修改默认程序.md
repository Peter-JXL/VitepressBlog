# 使用火绒拦截流氓软件修改默认程序

最近发现，在用网盘的时候，这些网盘总是会修改我的视频默认打开程序，或者修改默认看图软件等，烦不胜烦。

即使改回来了，过几分钟又 ™ 给我改回去了，试过几次都这样，气的我砸桌子 🤬。

网上搜了下解决方案，发现可以用火绒拦截这种流氓行为，下面讲解下步骤。

‍

## 一、安装火绒

火绒安全软件是一款小巧、强悍的电脑防护产品。

官网：https://www.huorong.cn/person   直接下载安装即可。

![](https://image.peterjxl.com/blog/image-20260310192458-ofu537j.png)

‍

## 二、配置拦截

安装后，打开主界面，选择左侧的「防护中心」：

![](https://image.peterjxl.com/blog/image-20260310194453-4ckjo5x.png)

‍

然后点击左侧的「系统防护」，再点击右侧的「系统加固」：

![](https://image.peterjxl.com/blog/image-20260310194525-135u3zt.png)

‍

点击注册表防护：

![](https://image.peterjxl.com/blog/image-20260310194611-vi4cbht.png)

‍

下拉找到「系统默认程序」这一项，然后点击左侧的开启按钮：

![](https://image.peterjxl.com/blog/image-20260310192910-i5t0gd7.png)

‍

## 最后

目前用了几天，发现能正常拦截了，身心都舒服了不少。

火绒还是比较良心的企业，会听取用户的意见。
