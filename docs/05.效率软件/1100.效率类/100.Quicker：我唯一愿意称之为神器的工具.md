# Quicker：我唯一愿意称之为神器的工具

![](https://image.peterjxl.com/blog/image-20250613102054-gzqdi2v.png)

一款我持续用了六年的工具。
‍<!-- more -->

## 为什么用 Quicker

举个例子，我之前是计算机专业的学生，几乎每天都要用电脑。

每天开机时，我都会有很多要打开的东西：刷题网站，文件夹，浏览器，Word 文档，文本编辑器... 等等一大堆东西。

久而久之，我觉得这也太麻烦了🥱。身为程序员，这些重复性的动作，就不能用程序来解决吗？

于是，我自己用 C#（一种编程语言）写了一个工具，里面填写每天要打开的文件和软件，并打包为一个 exe 可执行文件，每天开机后，只需双击 exe，就能一次性打开所有东西。

后来我就在想，这样也不太方便，每次添加、删除启动项都得改代码，重新打包，要不做个图形化界面？要不弄个开源项目，开始致富之路？🤓

可惜的是，在动手之前，我先去搜了下有无类似的工具 —— 因此发现了 Quicker。

‍

## 什么是 Quicker

一句话介绍就是：将日常操作封装为一个动作，后续只需一次点击即可完成所有操作。

再举个实际的例子：比如我想在哔哩哔哩搜索「烤鱼子」，在没有 Quicker 之前，我需要：

1. 打开哔哩哔哩
2. 在搜索框中输入关键字
3. 点击搜索

‍

而有了 Quicker 之后，我只需选中关键字，然后按下鼠标中键（打开 Quicker），单击动作（用 bilibili 搜索），就能一键完成上述步骤。

![](https://image.peterjxl.com/blog/image-20250613103212-pe7tems.png)

‍

![](https://image.peterjxl.com/blog/image-20250613104440-y6kvpfp.png)

‍

类似的，你想在任何平台一键搜索，都可以使用对应的动作。

---

Quicker 可以将多个日常操作（文本处理，翻译，录制，打开网址，打开软件，模拟按键...），封装为一个动作，是一个高效的自动化工具。

而且你甚至不用自己去封装动作，直接用广大网友封装好的动作即可。

> 如果你用过苹果手机上的「捷径」，你会发现 Quicker 就好比 Windows 版的「捷径」

本文后续会介绍 Quicker 的更多功能，手把手教你入门 Quicker。

‍

## Quicker 的历史

Quicker 于 2018 年底诞生，诞生之初功能就很完善、强大了，很多评测文章都将其誉为 2018 年度第一优秀工具。

我自己也在 2019 年初就开始使用，至今已经有 6 年了，几乎每天都在用，帮我节省了大量时间。

![](https://image.peterjxl.com/blog/image-20250613111803-wf5042a.png)

‍

虽然市面上已经有不少评测文章了，但感觉还是都不太适合新手，因此打算自己写一个保姆级教程，也督促我去更深入地去了解这个工具，用上更多功能。

作者的[回复](https://getquicker.net/Guides/Guide?id=bb29bbe9-31eb-42b5-235e-08d9afb372ec&step=d2cf76cf-1b14-4713-9f83-08d9afb37fb2)：

> 作者并不熟悉 mac 平台开发，且 windows 版仍有许多事情未完成，暂无精力转向 Mac；
>
> mac 平台有许多类似于 quicker 的效率软件，如：
>
> 快捷指令，Automator，AppleScript，alfred，raycast，hapigo，bettertouchtool 等
>
> Mac 对鼠标的支持不太理想。

如果你想在 macOS 上用，可以试试 [CirMenu](https://apps.apple.com/us/app/cirmenu/id6450661015)。

## 快速体验

接下来，就是打开 Quicker 了，我们先启动它（可以在开始菜单中找到 Quicker）。

> 注意：如果使用了 360 软件，[请将 Quicker 加入信任名单并且关闭驱动防护](https://getquicker.net/kc/help/doc/360-bug)。

打开后，默认要注册登录的，但我们可以先点击右下角的体验，先玩一下：

![](https://image.peterjxl.com/blog/image-20250613112208-j71i116.png)

‍

### 第一次运行动作

1. 按下鼠标中键（滚轮键），即可弹出快捷面板。
2. 点击 “记事本”, 这时候就会打开 Windows 记事本程序。

![](https://image.peterjxl.com/blog/image-20250613112357-f7bg43v.png)

‍

这个是 Quicker 的**最基础使用方法：打开面板，点击动作按钮执行动作。**

除了记事本，你可以在面板上自行添加任意软件、文件夹、网页等，后续就可以快速打开它们。

还可以打开一些特定界面：例如 Windows 的控制面板，系统属性等界面。

‍

### 选中内容执行动作

选中任意文字，然后打开面板，点击「快搜」：

![](https://image.peterjxl.com/blog/image-20250613113712-g7imblx.png)

‍

选择自己常用的搜索引擎后点击 “确定”（*有一些动作在首次运行时会显示配置界面，这些配置信息保存在本地计算机中*）：

![](https://image.peterjxl.com/blog/image-20250613113904-sy14t9g.png)

‍

可以看到 Quicker 使用浏览器打开了一个搜索页面，搜索关键词是刚才选中的文字：

![](https://image.peterjxl.com/blog/image-20250613113838-i2uhlq3.png)

‍

以上就是 Quicker 的最基本的运行方法。

## 禁用面板

如果在打游戏，玩到一半按了中键弹出了窗口该怎么办呢？

在「设置 - 弹出面板 - 黑名单」中，有一个**在全屏软件中禁用**的选项，可以添加 exe 文件，也可以在手动选择窗口。

这样就不用担心误触了。

## 关闭与钉住面板

如果误打开面板后，想要关闭面板，可以：

- 在面板窗口外面任意位置点击鼠标
- 或者点击面板右上角的 X 按钮
- 按键盘 ESC 按键也都可以隐藏面板

‍

如果你想钉住面板，可以点击面板右上角图钉符号：

![](https://image.peterjxl.com/blog/image-20250613144709-c6qkxu8.png)

‍

钉住后只能点击右上角的 X 按钮关闭面板。

## 自动切换面板

Quicker 的面板主要有上下两个部分组成，每部分上都有按钮区域。

上面是全局按钮区域，也是固定面板，面板上的动作不会随当前使用的软件而变化。

下方的的面板则是动态的，可以智能判断当前使用软件并给出推荐的快捷命令。

![](https://image.peterjxl.com/blog/image-20250613161623-gzc4zw4.png)

‍

例如，当你在 Excel 软件内打开面板时，可以看到下方展示的面板是不同的。

![](https://image.peterjxl.com/blog/image-20250613151159-4afj9bo.png)

‍

这是 Quicker 的一个重要功能：根据当前使用的软件自动加载动作页。

你可以点击 Quicker 菜单的 “场景与动作”：

![](https://image.peterjxl.com/blog/image-20250613151623-q6pwqmn.png)

‍

打开应用程序设置窗口，可以看到 Quicker 为哪些软件预先设置了动作页：

![](https://image.peterjxl.com/blog/image-20250613151836-dnw04b0.png)

‍

上下两个动作页的格子数量是固定的，**面板的每一个格子都可以放置一个动作**。

如果你觉得不够用，可以添加多一个动作页（有点类似在手机主屏幕上增加一屏）。

增加之后的动作页可以通过鼠标滚轮进行页面切换。

![](https://image.peterjxl.com/blog/image-20250613161843-n0tcjd3.png)

‍

你也可以在任意软件内打开 Quicker，会自动提示你是否需要添加动作页（否则显示默认动作页）：

![](https://image.peterjxl.com/blog/image-20250614114729-v8bkmf0.png)

‍

## 发现需求

如何发现自己需要的动作？

很简单，在日常中，如果你经常做某些操作，但同时步骤又比较复杂，就可以搜索相关动作。

例如：

- 我平时会看电影，看完后想看看豆瓣影评，就可以安装动作「用豆瓣搜」
- 平时想搜索某个文件，就需要复制选中的文字，打开 Everything —— 现在只需安装动作「用 Everything 搜」
- 想翻译某段文字，之前都得复制文字到翻译软件，现在只需用动作「翻译」
- 平时写博客，经常要压缩图片，之前都是将图片挪到压缩工具，现在只需用动作「图片压缩」
- 平时写代码，需要新建多个文件夹，现在只需用动作「一键创建模板文件夹」
- 想要将某个文件夹的内容挪到当前目录下，只需动作「解散文件夹」
- ......

文章开头举了个例子，提到我每天开机后要运行很多软件，此时我就可以用 Quicker 来帮我完成，我只需在开机后点击一下动作，然后就可以去做其他事情（例如刷牙、吃早餐等），做完后回来，电脑已经准备就绪了。

在工作后，每天开机都得打开 OA，邮件，编辑器等，此时一样可以用 Quicker 帮我们一键打开。

除了自己发现需求，你也可以在动作库中浏览，看看哪个动作惊艳到了你，可以给你提供很大的帮助。

## 新建和修改动作

除了用别人封装好的动作，你也可以自行新建动作。

在面板上右键，可以看到右键菜单很丰富：

![](https://image.peterjxl.com/blog/image-20250613154556-ts62h14.png)

‍

其中：

- 基础动作比较简单，一般是完成单一的任务，例如打开网址、发送文本、模拟按键等
- 组合动作比较复杂，一般包含数个基础动作
- 剩下的选项，顾名思义，比较简单，就不赘述了

‍

着重讲讲组合动作。我们点击它，可以看到这样的界面（黑字是说明）：

![](https://image.peterjxl.com/blog/image-20250613163520-0mrv5vt.png)

‍

右侧就是可供选择的基础动作，你可以拖动它们到中间进行编辑。

操作过程都很简单，我没看文档都会操作。

---

如果你觉得有点复杂，可以先看看其他人是怎么做的。我自己就是通过这样来学会哪些复杂动作的。

在动作上右键，选择编辑：

![](https://image.peterjxl.com/blog/image-20250613155836-veewgfm.png)

‍

就可以看到该组合动作内，是由几个基础动作构成的：

![](https://image.peterjxl.com/blog/image-20250613155906-oift9mk.png)

步骤解析：

1. 获取选中的文本：顾名思义，是 Quicker 帮我们获取到的
2. 如果 / 否则：会判断是否选中的文本

    1. 如果有选中文本：则会打开搜索网址，并拼到网址后面，完成搜索
    2. 如果没有选中：则直接打开 bilibili 搜索页面

‍

其他动作同理。你还可以直接修改他人的动作，使其符合自己的需求。
