# Chrome 扩展全指南（浏览器必备）

​![](https://image.peterjxl.com/blog/image-20250115162026-mwf9ptb.png)​

终于，我们来到了扩展的部分。

本文是《最全面的浏览器教程》第七篇，本文会介绍扩展的相关概念，安装方法，后续会持续更新一些好用的插件。

‍<!-- more -->

## 什么是扩展

关于扩展的概念，之前简单提了下，这里有必要详细说明。
‍
### 是什么

浏览器扩展，也叫浏览器插件，可以为我们的浏览器添加 **「额外的功能」** ，得到各种功能的 **「增强」** 。

本质上就是一些代码，你可以去安装插件来增加功能。
‍
### 有什么用

Chrome 有着 10w+ 的扩展应用，配合一些脚本，能实现你几乎所有想在浏览器上实现的功能，如广告拦截，密码管理，脚本管理，网页翻译，页面美化，强制夜间模式，网页长截图，录屏等。

网上一直流传着一句话：不装扩展（Extensions）的 Chrome 只能发挥它 20% 的能力。

各类实用的 Chrome 扩展是不少人选择 Chrome 浏览器的重要原因。
‍
### 代价是什么

Chrome 一个著名的缺点就是占内存。

​![](https://image.peterjxl.com/blog/image-20250105173550-77r74h3.png)​
‍

Chrome 会为每个扩展和标签页单独开一个进程：

​![](https://image.peterjxl.com/blog/image-20250105173602-5b2i6ln.png)​
‍

当然，这样做带来的好处是快速，高效。

总之，不要为了安装而安装，要装真正有用的插件。
‍
### 谁开发的

任何人都可以开发扩展：企业，团队，个人。

这也是为什么 Chrome 扩展生态这么好的原因，全球的人都可以为它开发插件，能不好才怪。

---

接下来，本文主要以 Chrome 为例，讲解插件相关的内容。
‍
## 打开扩展管理页面

​![](https://image.peterjxl.com/blog/image-20250109115047-laqch42.png)​
‍

扩展管理界面如下，可以搜索，停用，查看详情，移除扩展：

​![](https://image.peterjxl.com/blog/image-20250107111918-imfurge.png)​
‍

或者点击扩展管理图标，也能看到已安装的插件，并可以点击插件右侧的 `⁝`​ 对插件进行管理：

​![](https://image.peterjxl.com/blog/image-20250109115109-30i8seg.png)​

还可以在地址栏输入 chrome://extensions/ 进入扩展程序界面。

此外，点击详情：

​![](https://image.peterjxl.com/blog/image-20250118152852-lokubjz.png)​

## 安装扩展

总的来说可以分为两种方式：

* 在线安装：一些浏览器会有自己的扩展集市，可以访问它，然后搜索、安装扩展
* 离线安装：下载好扩展文件，拖放到浏览器进行安装
‍
### 在线安装

不同浏览器的扩展集市地址不同：

* Chrome：https://chromewebstore.google.com/category/extensions  （需魔法上网）
* FireFox：https://addons.mozilla.org/zh-CN/firefox/
* Edge：https://microsoftedge.microsoft.com/addons/Microsoft-Edge-Extensions-Home
* Safari：在 App Store 里搜索「扩展」，具体可参考 [在 Mac 上使用 Safari 浏览器扩展](https://support.apple.com/zh-cn/102343#install)
* Opera：https://addons.opera.com/zh-cn/ （需魔法上网）
* Vivaldi：同 Chrome 一样，可以直接打开 Chrome 商店页面进行插件安装
‍
也可以在浏览器的扩展管理页面，获取到扩展集市地址：

​![](https://image.peterjxl.com/blog/image-20250107114657-0y6ymil.png)​
‍

以 Chrome 为例，打开应用商店后，可以查看，搜索相关扩展：

​![](https://image.peterjxl.com/blog/image-20240201180029-ttl3fom.png)​
‍

找到扩展后，在扩展的详情页点击“添加至 Chrome” 即可安装：

​![](https://image.peterjxl.com/blog/image-20240201180154-thgtvau.png)​
‍

### 离线安装

主要步骤：

* 下载扩展文件（可以是 .crx 文件，也可能是 .zip 文件）
* 解压扩展（建议解压到一个平时不会很少动的文件夹，避免移动文件夹后导致扩展失效）
* 在浏览器开启「开发者模式」→ 加载已解压的扩展程序 → 选择刚刚解压的文件夹

下面分步骤进行讲解。

---

由于 GFW，没有魔法就无法访问到 Chrome 应用商店，也就无法下载扩展。

为此，国内有些网站搬运了 Chrome 应用商店的扩展，方便国内的用户下载并离线安装，例如：

* 极简插件：https://chrome.zzzmh.cn，干净简洁下载站，支持历史版本下载
* Crx4Chrome：https://www.crx4chrome.com，扩展数量应该是最多最全的，支持历史版本下载
* Crx 搜搜：https://www.crxsoso.com，实时解析各大扩展商店最新版扩展下载地址
* CrxDL：https://crxdl.com
* 扩展迷：https://www.kuozhanmi.com

---

下载后的扩展文件通常是 .crx 文件，其实就是一个压缩包，可以通过 [改文件后缀名](https://www.peterjxl.com/Windows/settings/useful/#显示文件后缀名) 来解压。也有些网站直接提供的是 .zip。

谷歌浏览器自 Chrome 67 版本开始，不支持将 CRX 扩展文件拖入浏览器直接安装了，读者可以自行验证：

​![](https://image.peterjxl.com/blog/image-20250107153509-e5ri7ys.png)​
‍

所以如果你下载到的是 .crx 文件，先改后缀：

​![](https://image.peterjxl.com/blog/image-20250107152217-g1k6nrr.png)​
‍

接下来，打开扩展管理页面，点击右侧开启开发者模式

​![](https://image.peterjxl.com/blog/image-20250107152357-c4gay1a.png)​
‍

打开发者模式打开后，会看到左上角的 「加载已解压的扩展程序」选项：

​![](https://image.peterjxl.com/blog/image-20250107152424-f1k1eyh.png)​
‍

点击该选项，就会打开文件夹选择框，选择刚刚解压的文件夹即可：

​![](https://image.peterjxl.com/blog/image-20250107152519-j59l66b.png)​
‍

### Chrome 插件伴侣

刚刚介绍的离线安装方式，要先解压，然后再打开扩展管理页面，加载文件夹，还是有不少步骤的。

为此，有网友开发了一款软件「Chrome 插件伴侣」，可以通过两次鼠标点击，轻松地，以无损，非破解的方式安装插件。

> 该软件目前找不到官网，但网上能找到很多下载地址，请读者自行搜索下载。
‍
​![](https://image.peterjxl.com/blog/image-20250107154043-2hmhavy.png)​
‍

如无必要，不建议勾选最下方的选项。

但安装成功后，扩展默认是禁用的，你可以打开扩展管理页面，手动启用：

​![](https://image.peterjxl.com/blog/image-20250107154117-6aia6w6.png)​
‍

或者打开 Chrome 后等几分钟，浏览器会弹出是否启用扩展程序的对话框，点击启用即可：

​![](https://image.peterjxl.com/blog/image-20250107154209-pso02v4.png)​
‍

### 离线安装的问题

离线安装扩展后，可能会遇到每次启动 Chrome 时，浏览器右上角会弹出「请停用以开发者模式运行的扩展程序」：

​![](https://image.peterjxl.com/blog/image-20250107165520-jmc0f2o.png)​

虽然并不影响使用，但每次都要关闭窗口的话，还是挺麻烦的。
‍
解决措施：

* 如果你以「请停用以开发者模式运行的扩展程序」为关键词进行搜索，你能找到很多解决方法：组策略，运行批处理，修改 dll 文件，修改注册表..... 这些方法一听就很复杂，实际上也是，还不一定有效，因为 Chrome 一直在更新
* 也可以通过弹窗拦截的方式，关闭弹窗，例如火绒。
‍
我的建议：还是在线安装好。可以先离线安装一些可以直连扩展集市的插件（在刚刚介绍的网址里找）：

​![](https://image.peterjxl.com/blog/image-20250108161216-cqtnng7.png)​
‍

注意，我个人建议还是尽量通过官方的扩展集市进行安装，避免下载到恶意扩展。
‍

安装完扩展后，这些扩展默认是不显示在扩展栏里的。

你可以选择是否固定到扩展栏：

​![](https://image.peterjxl.com/blog/image-20250109095352-871rgp2.png)​
‍

建议将一些经常修改配置的扩展，固定到扩展栏，方便使用。
‍
## 注意避雷

不要随便安装不可信的扩展，因为扩展权限很大。

在 [如何构建一个窃取一切的 Chrome 插件](https://mattfrisbie.substack.com/p/spy-chrome-extension) 这篇文章中，作者提醒大家注意，浏览器插件可以读取用户的所有信息，发到远程服务器，因此不要随便安装。她还提供了一个演示。

而且，已经有不少案例了：

* Stylish：一个用于自定义网页样式的知名扩展，超过百万用户，被软件工程师 [Robert Heaton](https://robertheaton.com/2018/07/02/stylish-browser-extension-steals-your-internet-history/) 发现存在窃取用户浏览历史、网页记录的行为，已被 Chrome 下架
* SuperCopy：一款能解除网页禁止复制功能的扩展，超 60 万人在用，Chrome 扩展商店精选插件，被 [V2EX 用户曝劫持流量](https://www.v2ex.com/t/1084504)，但目前还未下架，审核力度不太够啊🙃
* [用扩展进行挖矿](https://www.sohu.com/a/227226907_115128)
* ......

这种随着扩展用户量变多，被开发者卖掉，被后继者加点料的例子不少。

不单单是扩展，哪怕是开源项目也有类似的问题：[xz 后门的作者 Jia Tan 是谁？ - 阮一峰的网络日志](https://www.ruanyifeng.com/blog/2024/04/weekly-issue-296.html)

只能说，上网还是要小心啊。

## 最后

如果把手机比作浏览器，那么 APP 就相当于一个个插件。

没有安装 APP 的手机，只有基础的功能：接听电话，收发短信，拍拍照片等。

而安装 APP 后，就能上天入地，无所不能；虽然带来的代价是占用了内存和硬盘，但这值得。

希望大家都用上好用的扩展，提高效率，节省时间。
