# 900.谷歌逐步淘汰 V2 插件，广告拦截不能用了？

​![](https://image.peterjxl.com/blog/image-20250109154725-2n8tg6p.png)​

你的浏览器扩展可能一夜之间失效！快收下这份自救指南。

本文是《最全面的浏览器教程》第九篇，介绍目前扩展生态的问题。

‍<!-- more -->

## 起因

Chrome Manifest v2 是 Chrome 扩展程序的一种配置文件格式，是扩展的一部分，定义了一些扩展的特性，告诉 Chrome 扩展程序如何加载并执行。

2019，谷歌宣布推出 Chrome Manifest v3（简称 MV3），旨在提高扩展的安全性、隐私性、性能和可维护性，引入了新的架构和 API，有更严格的权限管理，加强了隐私性和安全性，能显著减少资源消耗。

谷歌是在强行推广 MV3 协议的：

* 后续不再受理 MV2 版本的扩展上架
* 获得精选徽章的扩展，如果不升级，会被删除徽章，这会导致搜索展现率大幅下降。
* 后续会自动停用 MV2 扩展（用户可以启用），并且不再允许从应用商店安装 MV2 扩展
* 后续将彻底禁用 MV2 扩展，无法再启用

​![](https://image.peterjxl.com/blog/image-20250109171218-ikt09hs.png)​
‍

## 影响

淘汰 Manifest V2 扩展之后，原有的扩展很多功能都不能用了，尤其是广告屏蔽类工具和脚本管理器等多个主流扩展程序，遭到了开发者和用户的强烈反对，谷歌推进的步伐被屡次放缓。

Chrome 插件的一大好处就是灵活性和有权限，结果一下就要把各种权限收掉...

​![](https://image.peterjxl.com/blog/image-20250109154649-ly50ue9.png)​
‍

虽然在公告中，谷歌提到目前扩展商店里超过 85% 积极维护的扩展程序，已经更新为 Manifest v3 版，但这其实是个文字游戏。

这里的 85% ，指的是「积极维护的扩展程序」中的 85%；而在商店里有几十万甚至上百万个扩展程序，但大部分扩展程序都已经不再积极维护，属于那种能用但没有功能更新的状态。

​![](https://image.peterjxl.com/blog/image-20250109171530-jyne2j7.png)​
‍

这不仅仅影响到 Chrome 浏览器，连带着所有使用 Chrome 内核的浏览器都会受影响。只要谷歌浏览器还是老大，他们就永远无法离开这个 Chrome 浏览器的生态系统。

可以说，该举会影响到整个浏览器生态圈。
‍
## 解决措施

1. 跑路：更换浏览器，例如 FireFox 会继续支持 MV2 扩展
2. 停更：备份扩展离线安装包 + 禁止浏览器更新（缺点：再也用不上新功能了）
3. 降级：如果已经更新了，考虑降级
4. 等待：如果开发者提供更新到 v3 版就好了。但这取决于开发者，而且很多功能可能不能用

## 个人看法

自 MV3 发布以来，谷歌就隐私机制发布了几项有争议的 “隐私沙盒” 提案，这些提案会在万维网联盟 (W3C) 中进行讨论。

虽然从技术上讲，“任何人” 都可以听取公开会议的意见，但只有 W3C 成员才能提出有关规范的正式文件并担任领导职务；想要成为 W3C 成员，就需要金钱和时间。

对大型跨国公司而言，金钱和时间都不是问题，但对于以用户为中心的小团体来说，这可能是就是巨大的障碍了。

不解决投票权的问题，那么市场份额越大的，话语权就越大。

无论这个提案是的如何的善意，但现实却是数以百万计的人的互联网上体验，取决于大公司和标准组织中少数人的道德规范。

可能该提案确实能提升用户体检，但我不太相信谷歌。

我更相信一句话：命运赠送的礼物，早已在暗中标好了价格。
