# 600.互联网冲浪必备脚本

在网上冲浪，少不了使用一些大型的论坛和平台，装一些相关脚本也是有必要的。

本文就介绍一些针对大型互联网平台的脚本。

‍<!-- more -->

## B 站

[Bilibili-Gate](https://greasyfork.org/zh-CN/scripts/443530-bilibili-app-recommend)：将网页端的推荐机制，修改成 APP 版的推荐机制，类似的脚本还有 [首页推荐](https://github.com/indefined/UserScripts)

[抽奖动态删除](https://greasyfork.org/zh-CN/scripts/387046)：可以一键删除抽奖动态，还可以选择是否取关对应的 UP 主

[B 站屏蔽增强器](https://greasyfork.org/zh-CN/scripts/461382)，[Bilibili 按标签、标题、时长、UP 主屏蔽视频](https://greasyfork.org/zh-CN/scripts/481629) ：可以屏蔽视频、屏蔽动态、屏蔽评论，按关键字屏蔽、按用户名屏蔽、按 UID 屏蔽、按标签屏蔽、按标题屏蔽、按时长屏蔽，只要你想，甚至可以用它们拉黑整个 B 站。

[Make BiliBili Great Again](https://greasyfork.org/zh-CN/scripts/415714-make-bilibili-great-again)：自动禁用掉劣质的 PCDN，让网络体验更好；去除充电列表展示，去除 URL 上的跟踪参数，动态首页宽屏优化......

[收藏夹 Fix](https://greasyfork.org/zh-CN/scripts/383143)：收藏的视频如果被删了，就会什么都看不到，标题、封面，甚至连 UP 主是谁都不显示。用了这个脚本，就可以看到失效视频的基本信息，播放量、收藏数、UP 主、投稿时间、标题简介，点击一下还能看到更详细的数据（封面等）

[Bilibili-Evolved](https://github.com/the1812/Bilibili-Evolved)：已有 23k Star，B 站的全功能工具箱，采用插件机制，可以安装很多功能，例如简化首页，简化评论区，去掉视频弹窗，去除右侧推荐视频，支持下载视频、封面、弹幕、音频，优化专栏复制体验，屏蔽 UP，设置夜间模式，自动点赞......
‍
以下是我自己用的一些小脚本：

* [get bilibili cover ](https://greasyfork.org/zh-CN/scripts/30714)：在视频列表上以及视频播放页面，按 Ctrl + 鼠标右键 ，就会在新窗口打开封面图
* [下载 Bilibili 动态页面图片](https://greasyfork.org/zh-CN/scripts/421885-bilibili-download-pictures)：下载 “Bilibili 动态” 时间线页面的图片，也可下载视频（720P 单文件）
* [哔哩哔哩图片打包下载](https://greasyfork.org/zh-CN/scripts/402693)：下载 B 站 UP 主 Bilibili 动态相册相簿图片，以及视频封面，专栏图片和 UP 主头像以及主页壁纸，直播间封面和直播间壁纸

## 知乎

* [知乎美化](https://greasyfork.org/zh-CN/scripts/402808)：非常强大的脚本，修改了很多的页面布局，添加信息流标签（问题，文章，视频等），我觉得装这一个脚本就能满足大部分需求了
* [iZhihu](https://greasyfork.org/zh-CN/scripts/350-izhihu)：本着「我爱知乎」的原则， 实现了首页动态过滤选项、自动展开「折叠区」、导航栏随页面卷动自动收起、搜索结果「在新页面打开」等，提供了美化选项（优化样式外观、将回答者信息挪至回答的下方、在页面右侧浮动弹出评论列表及输入框等），同时提供  [Chrome](https://chrome.google.com/webstore/detail/izhihu/omcldpfdihfogiklcdlopeokkedbhjop)、[Firefox](https://addons.mozilla.org/zh-CN/firefox/addon/izhihu/)、[Safari](http://izhihu.unogz.com/safari/iZhihu.safariextz) 扩展版本。
* [知乎备份剪藏](https://greasyfork.org/zh-CN/scripts/486538)：将喜欢的知乎回答 / 文章 / 想法保存为 markdown /zip/png，支持评论保存，挺实用的一个脚本，作者非常用心，文档写的很详细，更新也很频繁
* [知乎去除链接跳转](https://greasyfork.org/zh-CN/scripts/369514)：暴力替换知乎页面的链接，去除知乎重定向，直接跳转到页面
* [知乎屏蔽盐选](https://greasyfork.org/zh-CN/scripts/422339)：如题，老是会看到一些盐选内容，还是挺烦的
* [【知乎优化助手】](https://greasyfork.org/zh-CN/scripts/410221)：屏蔽知乎广告、其他功能优化
* [不上知乎](https://greasyfork.org/zh-CN/scripts/376027-nozhihu)：在访问知乎的时候跳到 Quora。挺搞的这个脚本 🤣

注意：你可能会看到不少个人使用免登录的脚本，但知乎后续做了不少限制，很多内容必须登录后查看，因此这里就不列出了。

## 百度网盘

[网盘链接检查](https://greasyfork.org/zh-CN/scripts/394216)：自动识别并标记百度云、蓝奏云和腾讯微云的链接状态

[全载](https://greasyfork.org/zh-CN/scripts/22590)：一键将下载链接资源离线保存至网盘。在磁力链接、视频、压缩包或其他下载链接旁添加按钮，点击自动离线下载至各大网盘（或在线云点播，磁链转种子）。支持百度云网盘、115、Furk、Seedr、迅雷离线、Pcloud、小米路由器等。

[网盘自动填写提取密码](https://greasyfork.org/zh-CN/scripts/1002)：将网盘地址 # 后面跟着的字符（一般是密码），作为提取码自动输入

[百度網盤自動轉存](https://greasyfork.org/zh-CN/scripts/19864)：自動轉存當前文件至上次轉存路徑

[网盘自动填写访问码【威力加强版】](https://greasyfork.org/zh-CN/scripts/29762)：自动提交密码。记录网盘信息，再次打开该分享时自动填充。

## 豆瓣资源下载大师

豆瓣目前在国内还算是评价电影质量的标杆，很多人会根据豆瓣评分来判断质量。

但豆瓣仅提供在线观看的视频（甚至没有），很多时候看到不错的电影，还得自己去找资源下载。

​![](https://image.peterjxl.com/blog/image-20250207103223-gh1mzw2.png)​
‍

叠个甲，关于我为什么很少在线看：

* 一来国内很多平台都有可能会删减镜头、改结局。举个例子：[2024 内地院线进口片删减情况小结](https://mp.weixin.qq.com/s/GhVKDe1kYDF7mJypVTQj-A)
* 二来是付费，让我付费看阉割版的，我才不干呢 😪。
* 三来是网速问题，网不太好，如果看电影遇到一次卡顿，是很糟心的...... 而本地看还能很方便地前后拖动进度。
‍
推荐一个脚本：[豆瓣资源下载大师](https://greasyfork.org/zh-CN/scripts/329484)，几乎是完美地解决了该问题。

​![](https://image.peterjxl.com/blog/image-20250207110653-orzych3.png)​
‍

安装脚本后，在豆瓣打开一个电影详情页，该脚本就会帮你去海量资源下载站寻找，是否有该资源，并且在右侧栏显示：

​![](https://image.peterjxl.com/blog/image-20250207110101-t46xo23.png)​
‍

不仅仅是影视资源，几乎所有豆瓣条目都可以，例如电视剧、纪录片、综艺、动画、音乐、图书等等，都会帮你找资源。

还有个重点功能得介绍下，豆瓣上有很多观影达人整理的豆列，里面会对某个特定主题的推荐同类的精品，比如「十部最经典的二战电影」「北美电影票房总排行榜」等等。所以找到了一个好的豆列，基本上可以顺藤摸瓜把该主题的经典影片看个遍。

但豆瓣一直没上线豆列搜索功能，好在该脚本加上了：

​![](https://image.peterjxl.com/blog/image-20250207110415-x98xn7g.png)​
‍

它还会给一些豆瓣条目，主要是以国外电影为主额外添加了 IMDB 评分、 Metascore 评分、烂番茄评分三种评分形式（相当于美国的电影豆瓣评分）：

​![](https://image.peterjxl.com/blog/image-20250207112142-ucyhs50.png)​

* IMDB 评分是网民的评分，满分和豆瓣评分一样都是 10 分
* Metacritic 是专门收集对于电影、电视节目、音乐专辑、游戏的评论的网站。上面的 Metascore 评分是传统媒体专业评价为主，满分 100 分。
* 烂番茄评分对一部影片的评价是基于所有网站认证影评人的评价得出，网站会根据影评人的评价认定是好「新鲜番茄」 还是不好「烂番茄」 ，烂番茄新鲜度指数是百分比越高代表越好，100% 的意思是，100 个影评里有 100 个好评，这个百分比就是好评率。
‍
关于豆瓣的增强脚本有很多同类，但这个绝对是最值得用的一个，非常受欢迎，发布后一个月破万安装量。

其他关于豆瓣的脚本：

* [绿豆瓣](https://greasyfork.org/zh-CN/scripts/401525)：豆瓣电影 BT / 种子 / 资源 / 磁链 / 字幕 一键搜索下载，在线观看
* [豆瓣导出工具](https://greasyfork.org/zh-CN/scripts/401926)：导出豆瓣上看过和读过、想看和想读的列表。
* [豆瓣读书资源助手](https://greasyfork.org/zh-CN/scripts/371723)：在豆瓣读书页面展示资源站下载链接
* [MoreMovieRatings](https://greasyfork.org/zh-CN/scripts/7687)：豆瓣和 IMDb 互相显示评分，还会在豆瓣上显示烂番茄评分、Metascore

## GitHub

[Github 增强 - 高速下载脚本](https://greasyfork.org/zh-CN/scripts/412245)：这个脚本能让下载速度提供 N 倍，还是很不错的，作者是出于公益的态度进行维护，支持。

安装之后，它就会为 Github 的 Clone、Release、Raw、Code (ZIP) 添加高速下载地址，而且还不止一个：

​![](https://image.peterjxl.com/blog/image-20250206212225-qoy5j8n.png)​
‍

可能有小伙伴会好奇原理，开发者解释：

> 目前有很多 Github 文件下载加速的网站或开源项目，脚本只是将加速后的文件下载地址添加到了网页，省去了手动获取的麻烦，方便直接点击高速下载！
>
> 只要你链接这些网站的速度比 Github 原生用的亚马逊 CDN 的速度更快，就是加速下载！
‍
[GitHub 汉化插件](https://openuserjs.org/scripts/52cik/GitHub_汉化插件)：很多新手朋友不太会玩 GitHub，可能被全英文界面所困扰，这款脚本实现汉化了 GitHub 界面的部分菜单及内容，新手熟悉之后可选择停用脚本恢复英文模式。

[Github News Feed Filter](https://openuserjs.org/scripts/jerone/Github_News_Feed_Filter)：在 Github 首页添加一个信息流的过滤器，帮助用户按照 Commits、Repo、Starred、Wiki 的分类显示不同的关注信息流。

## 学术

在国内，总会遇到写论文的场景（最常见的就是毕业论文了）。

关于下载论文，如果是校园网，有时候还可以免费下载；否则就比较麻烦了，得花不少钱。

推荐一个脚本：[学术论文期刊免费下载工具](https://greasyfork.org/zh-CN/scripts/492146)，支持众多学术网站的下载。

原理：调用外部接口，实现免登录对文章进行下载。

在安装好脚本后，在论文详情页里能找到下载按钮，点击即可一键下载。

​![](https://image.peterjxl.com/blog/image-20250206193706-vfwm3ec.png)​
