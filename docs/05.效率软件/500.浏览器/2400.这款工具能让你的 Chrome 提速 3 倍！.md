# 这款工具能让你的 Chrome 提速 3 倍！

之前的文章，介绍了很多浏览器的使用技巧、扩展推荐、脚本安利。

但有一点没提到：那就是 Chrome 的运行速度优化。

‍<!-- more -->

## 问题

Chrome，是浏览器的扛把子，运行速度毫无疑问是很快的。

但随着使用时间的增加，运行速度会肉眼可见地变慢。这是因为，Chrome 浏览器内的 SQLite 数据库会随着使用时间的增加产生大量碎片，所以启动使用这类数据库的应用程序就需要额外时间来处理，这样浏览器总体的响应速度就会受影响。

我们可以使用一些清理工具，提高运行速度。
‍
## SpeedFox

​![](https://image.peterjxl.com/blog/image-20250214152740-237zeb8.png)​

官网：https://crystalidea.com/speedyfox

功能：对浏览器进行优化，在不丢失任何数据的前提下压缩这些数据库，从而提高浏览器的运行速度。

支持：Windows（几乎所有常见的浏览器） 和 Mac（仅支持 FireFox），具体可以在官网看到

效果：按照官方的说法，通过 SpeedyFox 优化后，浏览器启动速度可以加快 3 倍。

​![](https://image.peterjxl.com/blog/image-20250214152935-1ffbxgk.png)​
‍

除了启动速度，在打开网页以及执行菜单操作上面均有不同程度的提升，指哪打哪，流畅如初。
‍
## 使用方法

在官网下载压缩包（仅 732 KB，解压后也只有 1.5 M）

​![](https://image.peterjxl.com/blog/image-20250214153207-te63u5b.png)​
‍

以 Windows 为例，解压后会得到一个 exe 文件，双击即可运行：

​![](https://image.peterjxl.com/blog/image-20250214153322-04euxxg.png)​
‍

运行后， 它会自动识别 Firefox 和 Chrome 浏览器以及以 Chromium 内核为内核的浏览器配置文件路径，勾选需要优化的浏览器，然后点击「Optimize」优化就可以了。

​![](https://image.peterjxl.com/blog/image-20250214153511-0zn6za4.png)​
‍

注意：如果是正在运行的浏览器，是无法被优化的，请先关闭浏览器。

​![](https://image.peterjxl.com/blog/image-20250214153547-ngcnyct.png)​
‍

可以看到优化速度还是很快的，几秒内即可完成：

​![](https://image.peterjxl.com/blog/image-20250214153634-9h83gwq.png)​
‍

至于效果，我目测的确有很大的提示，但很难用数据展示出来，就不说了。
‍
## 手动优化

SpeedFox 可以优化大部分以 Chromium 为内核的浏览器，也就是所有国产浏览器都行。

所以，如果有没识别出来的浏览器，可以手动添加。

由于我没怎么用国产浏览器，这里还是以 Chrome 作为演示，但步骤都是一样的。

打开要优化的浏览器，在地址栏中输入 `Chrome://version`​ 并回车，会打开当前浏览器的相关信息，例如版本，文件路径等：

​![](https://image.peterjxl.com/blog/image-20250214154436-ruygru8.png)​
‍

我们复制「个人资料路径」的信息，后面的 Default 不要复制，`C:\Users\peterjxl\AppData\Local\Google\Chrome\User Data`​

然后在 SpeedyFox 菜单栏 Add custom profile：

​![](https://image.peterjxl.com/blog/image-20250214160224-8xz3php.png)​
‍

然后在新弹出的界面里，输入刚刚复制的个人资料路径：

​![](https://image.peterjxl.com/blog/image-20250214160236-ikxox39.png)​
‍

添加之后它会自动识别，然后就可以像刚刚介绍的步骤一样，去优化该浏览器了。
‍
## 最后

关于优化频率，官方建议：根据自己使用浏览器的时间，每 1-2 周优化一次浏览器。

当你觉得浏览器变慢了，除了重装，还可以试试这个 SpeedFox，简单快捷，同时不影响浏览器本身的数据（历史记录，密码，书签等）。
