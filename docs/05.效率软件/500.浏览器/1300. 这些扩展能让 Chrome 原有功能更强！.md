# 1300.这些扩展能让 Chrome 原有功能更强！

Chrome 是很好用，但还不够好用，特别是它自带的一些功能，所以扩展集市里有一堆针对原有功能的增强。

<!-- more -->

例如：

* 标签页相关：双击关闭，双栏，侧边栏，自动冻结，收纳，预览，拖拽打开，批量打开...
* 扩展管理：自动根据网站加载/停用扩展，一键启用/关闭扩展
* 历史记录：支持按日分类、统计、高级搜索......
* 书签整理：页面美化，同步书签，加密书签......
* ......

## 标签页双击关闭

Chrome 并未提供双击关闭标签页的功能。在打开网页后，想要关闭标签页，要点击选项卡上的 x 按钮。

如果打开网页少的话倒还好，但是如果网页多的话，那就会非常麻烦，甚至还会点错以致关错网页。

虽然快捷键 Ctrl +  W 也不错，但有更多选择总是好的。

这个情况，也不能说是大问题，就是小缺点吧，毕竟，就连很多国产浏览器，都支持双击关闭标签的功能 👀

我们可以用 [Double Click Closes Tab](https://chromewebstore.google.com/detail/double-click-closes-tab/megplcpdkmjjoondippkedoaidkeikcm)：

​![](https://image.peterjxl.com/blog/image-20250120174827-saan02z.png)​
‍

安装后，有多种关闭标签页方式：

* 单击扩展图标
* 左键双击网页上的任意空白处
‍
如果你觉得双击容易误关，还可以修改配置，先进入设置页面：

​![](https://image.peterjxl.com/blog/image-20250120175232-79lom2q.png)​
‍

可以改为 “三次单击” 或 “双击右键”：

​![](https://image.peterjxl.com/blog/image-20250120175313-w6hh26k.png)​
‍

之前介绍浏览器快捷键时说过，Ctrl + Shift + T 能快速恢复误关闭网页。

这个扩展同样提供了类似的功能：Shift + 双击网页空白处，即可恢复刚关闭的网页。

如果你想将 Shift 键改成其他的，也是可以的：

​![](https://image.peterjxl.com/blog/image-20250121165352-lgfqlw6.png)​
‍

该扩展的其他功能我感觉没啥用，感兴趣的同学可以自行尝试。

双击关闭这个功能看起来是小事，但能提高效率，就是好东西，值得安利。
‍
## 标签页复制

[TabCopy](https://chromewebstore.google.com/detail/micdllihgoppmejpecmkilggmaagfdmb)：可以一键复制网页的标题和网址，可以很方便的分享给他人。

支持两种模式：

* 简单模式（Simple Mode）：单击图标复制当前标签页，双击复制当前窗口标签页，三击复制所有窗口标签页
* 花式模式（Fancy Mode）：点击扩展图标后会弹出一个窗口，你可以自己选择复制网页链接的范围和格式。

例如，支持复制为 Markdown 格式（我经常用），或者「标题+网址」、JSON、HTML 等格式。

​![](https://image.peterjxl.com/blog/image-20250202181042-ba78idi.png)​

以前，分享网页时，发送个网址，别人通常只有打开了才知道 🙄。单纯一个网址，没有任何有效信息。

现在，再给别人发链接的时候，可以用这个扩展，复制网页链接时会自动加上标题，让对方不用打开链接也能知道具体是什么内容，让对方觉得你很细心，留下好印象。与人方便，与己方便。

类似扩展：

* [CopyTabTitleUrl](https://chromewebstore.google.com/detail/copytabtitleurl/lmgbdjfoaihhgdphombpgjpaohjfeapp)：TabCopy 的平替
* [Copy All Urls](https://chromewebstore.google.com/detail/copy-all-urls/djdmadneanknadilpjiknlnanaolmbfk)：复制网页的 URL（支持文本，HTML，JSON 等格式）
* [二维码生成器](https://chromewebstore.google.com/detail/lkdokbndiffkmddlfpbjiokmfkafmgkm)：300w 用户，点击扩展即可生成当前网址的二维码，同时可以主动输入网址生成二维码

## 标签页侧边栏

目前浏览器的标签页，大部分都是在浏览器顶部的（也叫水平选项卡），如果一口气打开很多，容易看不清：

​![](https://image.peterjxl.com/blog/image-20250120162302-dsss72g.png)​
‍

之前介绍过分组功能，能起到一个分类的作用，缓解一下问题。

但有个更好的方案：将标签页挪到侧边栏，看起来更直观。

其实这个功能在十几年前就有了，但当时 Chrome 做的不好，后来被取消了。

​![](https://image.peterjxl.com/blog/image-20250120162947-m7a951z.png)​
‍

直到后来，很多浏览器支持侧边栏的垂直标签，因此在 2023 年，Chrome 114 发布时，谷歌把侧边栏的开发权限给了开发者，Chrome 有了新的选择，例如 [VSpace](https://vspace.cyhuajuan.site/)，提供侧边栏的垂直书签和标签页管理器。

​![](https://image.peterjxl.com/blog/image-20250120163820-ahiy53x.png)​
‍

安装后，点击扩展图标即可唤出垂直标签，或者用快捷键「Alt + Q」，大概长这样：

​![](https://image.peterjxl.com/blog/image-20250120165904-l7nuogw.png)​
‍

可以用鼠标拖动标签页，调整位置；鼠标悬浮在页面上，会出现关闭按钮。

VSpace 提供了一个树状结构，同一个页面打开的新标签页，就会归到下一级，方便归类管理：

​![](https://image.peterjxl.com/blog/image-20250120164547-xby6nyb.png)​
‍

既然用分上下级了，自然而然就诞生了一键关闭所有下级的功能：

​![](https://image.peterjxl.com/blog/image-20250120165703-5y9py7t.png)​
‍

网上冲浪时，经常会在一个网站打开多个子页，看完后直接关闭上一级即可，非常方便。

此外，它还支持同步，新建多个面板（例如工作一个，学习一个），设置快捷键等。

更多操作请看官网文档，也可看看网黑哥的文章：[Chrome 重新伟大！](https://mp.weixin.qq.com/s/oD75MTxxsHxfVlLb_7vt2A)

类似的扩展：

* [书签侧边栏](https://chromewebstore.google.com/detail/jdbnofccmhefkmjbkkdkfiicjkgofkdh)：30w+ 用户，4.5 星，精选插件
* [Tabs Outliner](https://chromewebstore.google.com/detail/tabs-outliner/eggkanocgddhmamlbiijnphhppkpkmkl)：10w + 用户，4.5 星，精选插件
* [侧边栏垂直标签页](https://guokai.dev/)：6w+ 用户，4.5 星，精选插件
* [tabVertikal](https://chromewebstore.google.com/detail/aahjeignibghcnabifeoclpebaleldkj)：3k+ 用户，4.4 星，精选插件
* [Tab Manager in Side Panel](https://chromewebstore.google.com/detail/tab-manager-in-side-panel/dhdfjphlghgaokjflinlfcacdafpmiom)：1k+ 用户，4.5 星，精选插件
‍
当然，如果你平时打开的标签页比较少，或者屏幕不够大，那么不用该扩展也是正常的，不要为了用而用。

此外，Arc、Edge 和 FireFox 已经内置了垂直标签功能。

## 标签页收纳

​![](https://image.peterjxl.com/blog/image-20250130170654-1t87exk.png)​

[Better-Onetab](https://chromewebstore.google.com/detail/better-onetab/eookhngofldnbnidjlbkeecljkfpmfpg)：当标签页多的时候，除了分组，用扩展冻结内存，还可以将他们收纳到一个列表，等用到的时候再全部打开。

例如，安装后，你可以鼠标右键单击扩展，选择储存当前所有标签页：

​![](https://image.peterjxl.com/blog/image-20250130171138-rfhov9o.png)​
‍

随后，Better-OneTab 会将所有标签页记录到一个列表，然后关闭这些标签页。

之后，你可以鼠标左键单击 Better-OneTab，能看到其帮你记录的标签页，并可以恢复（重新打开这些标签页）：

​![](https://image.peterjxl.com/blog/image-20250130171457-6b51pwj.png)​
‍

这个扩展还是挺好用的，例如我关机前会将所有标签页储存，后续再重新恢复。

而且这个扩展，比起标签页分组、内存自动冻结更简单粗暴，收纳后更简洁，页面更干净，我喜欢。

该扩展基于 [OneTab](https://chromewebstore.google.com/detail/onetab/chphlpgkkbolifaimnlloiipkdnihall?utm_source=chrome-ntp-icon)，作者自述：OneTab 扩展还缺少一些诸如简易列表，只储存选中的标签页等功能。由于开发时间久远且长期无人更新，并且也不开源，所以也没办法为 OneTab 增加新的功能，因此这里又做了一个更好的（Better-Onetab）扩展。它比 OneTab 拥有更美观的界面以及更多的功能。

## 扩展管理

少开点标签页、冻结标签页，是简单有效降低内存占用的好方法。

但此外，扩展也是会占用内存的，除非你停用它，而且停用扩展比关个网页麻烦多了。

​![](https://image.peterjxl.com/blog/image-20250130175416-ikdntf6.png)​
‍

为此，一些管理扩展的扩展出现了——例如二管家，免费开源，颇具好评，用户有 1w+：

​![](https://image.peterjxl.com/blog/image-20250119163355-ggxjppe.png)​

二管家最核心的功能是：根据规则自动开关扩展

比如「B 站下载助手」这个扩展，其实我们只需要它在 B 站上运行，只需在设置里配置下即可。

同理，像是比价扩展仅需要在购物网站上开启，翻译扩展可能只需要在自己常上的国外网站上启用，图片批量下载扩展只需要在常去设计网站上使用...

根据自己对一些扩展的使用需求，你可以尽量减少很多扩展在其用不到的时候运行，对于有大量扩展的同学来说，必备。

其他功能：

* 导入/导出配置，可以很好地进行备份
* 一键关闭开启单个或多个扩展
* 自动推荐扩展
* 导出扩展 crx 文件
* ......更多细节，可以看看网黑哥的文章：[这才是 Chrome 扩展爱好者真正必装好东西](https://mp.weixin.qq.com/s/G7SVhP0MZoRTdAapKjFczw)

个人认为，既然没说存在恶意代码，则大概率不是这方面的问题，而且是开源的，还是可以一用的，可以在我的《Chrome 扩展全指南》中提到的网站下载离线版。

Chrome 默认的历史记录功能是比较弱的，只有一个搜索功能，一旦忘记要查询的网页名字，那就惨了，得按照时间线一点一点往下翻，很容易错过。：

​![](https://image.peterjxl.com/blog/image-20250119170436-waqlm5x.png)​
‍

所以  Chrome 商店 上关于优化历史记录的扩展就有很多（可见用户的需求都是相同的 🙃），例如 [History Manager](https://chromewebstore.google.com/detail/history-manager/ifekbbngjjcachfekcpeejphmkeckged)：

​![](https://image.peterjxl.com/blog/image-20250119170602-k6ukth7.png)​

* 支持统计数据并使之可视化（生成图表）

## 下载管理

按下 Ctrl + J 会打开 Chrome 默认的下载管理器，非常的简陋，想看下载进度或管理文件都不方便：

​![](https://image.peterjxl.com/blog/image-20250202170406-1cajivj.png)​
‍

这里推荐 [Chrono下载管理器](https://chromewebstore.google.com/detail/mciiogijehkdemklbdcbfkefimifhecn)，有 70w 安装量，最好用的 Chrome 第三方下载管理器，是精选扩展。特点：

* 无缝接管浏览器下载功能，
* 拥有一个直观好用的下载任务 / 文件管理器
* 提供了非常好用的「**资源嗅探器**」，可以帮你检测并提取网页上所有的链接、图片、音频和视频等内容，还可以根据文件类型、使用正则表达式来筛选出需要批量下载的地址，轻松搞定批量下载。

​![](https://image.peterjxl.com/blog/image-20250202172258-kn6q0iv.png)​
‍

相关扩展：

* [IDM](https://www.internetdownloadmanager.com/)：Internet Download Manager，老牌下载神器，在下载界可谓家喻户晓，支持嗅探功能，支持从各种网站下载文件，并通过智能文件分割技术提高下载速度。付费软件。
* [NDM](https://neatdownload.com/)：Neat Download Manager，媲美 IDM 的下载工具
* [FDM](https://www.freedownloadmanager.org/)：Free Download Manager，类似 IDM 的竞品
* [猫抓](https://chromewebstore.google.com/detail/jfedfbgedapdagkghmgibemcoggfppbb)：专门解析下载 m3u8 视频的扩展，推荐阅读网黑哥的[文章](https://mp.weixin.qq.com/s/9x1CYrMqNZ9AkXG_2eztfw)
* [CocoCut](https://cococut.net)：支持视频嗅探，对于一些能播放但无法下载的视频，提供把网页挂在后台录屏的下载模式

## 自动隐私模式

​![](https://image.peterjxl.com/blog/image-20250119203552-p6lfgcs.png)​

浏览历史属于比较隐私的内容，通常没有人会想被他人看到。

虽然浏览器有无痕模式，但相信大部分人都没有使用它的习惯，通常是事后才想起来删记录。

最理想的方式：打开特定网站时，浏览器能自动用无痕模式打开，然后这些网址还得隐藏在收藏夹里，只有我能看到。

为此，推荐一个扩展，能完美解决这个问题：[auto-incognito](https://chromewebstore.google.com/detail/auto-incognito/iligdhpfclclkdegfdicjniagankbpdb)，自动隐私模式

​![](https://image.peterjxl.com/blog/image-20250119203842-nvie0qd.png)​
‍

它的功能非常简单。当你打开某些网站的时候，点击扩展图标，会有一个「添加当前标签页」的按钮：

​![](https://image.peterjxl.com/blog/image-20250119204538-6uk9j1v.png)​

这里的添加是指添加到黑名单，点击后，会直接进入隐私模式，同时历史记录也会被删除，在浏览器中也查不到。

你也可以手动输入网址，然后点击「添加到黑名单」。

如果不想让其他人看到黑名单怎么办呢？别担心，它还有一个设置密码的功能：设置后，每次查看黑名单都要输密码。

有了这个扩展，很方便测试人员，例如在一个网站登录多个账号进行测试的情况。

也适合用来打开一些个人喜欢的网站。其实这些网站也并不一定见不得人，但总得有点自己的空间是吧？

相关扩展还有：

*  [Go Incognito](https://browsernative.com/go-incognito/)：20w用户，通过快捷键，快速跳转隐私模式
* [InCognito](https://chromewebstore.google.com/detail/incognito/kencjkgapindpgehbgolojoocgpcepfk)：3w 用户，可以很方便地匿名浏览 LinkedIn 个人资料
‍
---

目前浏览历史记录的问题解决了，那如果每次打开一些网站都输手动入网址，太麻烦了。如果将其收藏在书签，那和不删历史记录也没太多区别。

同样的，书签也有类似的扩展，可以对书签进行加密。

## 书签管理

​![](https://image.peterjxl.com/blog/image-20250119210416-ay2oe9n.png)​

官网：https://bookmarks.mengxianglong.com

扩展：https://chromewebstore.google.com/detail/jikanmlokejbknkfemnbhaebmmibgmmb

安装后，点击扩展图标，能看到六大功能选项：

​![](https://image.peterjxl.com/blog/image-20250119211609-h2inrwj.png)​

* 第一个图标是默认书签栏，这里和浏览器默认书签是一致的。
* 第二个是置顶功能，可以将常用书签置顶到这，方便后续访问
* 第三个能看到最近访问的书签
* 第四个能看到最常访问的书签
* 第五个能看到最近添加的书签
* 第六个能看到加密书签（需要先设置密码）
‍
我们可以将鼠标悬浮到书签上，就能看到一个锁的按钮，点击之后就可以这个书签给加密了。

​![](https://image.peterjxl.com/blog/image-20250119212200-j20au14.png)​
‍

除此之外，它还可以搜索书签，标签管理，同步书签，切换深色/浅色主题，还有专业版功能（需付费）：

​![](https://image.peterjxl.com/blog/image-20250119213236-j43m1el.png)​
‍

说实话，就算没有这个书签加密功能，这个扩展也比浏览器默认的书签功能好用多了。

我发现关于书签管理的扩展，有非常多，可能该功能更常用吧：

* [Bookmarks Artist](https://github.com/liujuntao123/bookmarks-artist)：提供一个查看浏览器书签的全新界面（挺好看的），开源
* [懒猫书签](https://github.com/Alanrk/LazyCat-Bookmark-Cleaner)：清理失效书签、重复书签和空文件夹，开源，本地运行，不收集数据
* [MarkMagic](https://markmagic.top)：智能书签管理解决方案，一键扫描、清理失效和重复链接
* [潮汐收藏](https://www.tidemark.cc/)：搜索速度快，支持云同步，跨浏览器同步书签，数据导入/导出，相关介绍请看 [这篇文章](https://mp.weixin.qq.com/s/bv2_8FCLlO26dSSkklRA0Q)
* [Floccus](https://chromewebstore.google.com/detail/floccus-bookmarks-sync/fnaicdffflnofjppbagibeoednhnbjhg)：支持跨浏览器同步书签，4w+ 用户，精选插件，操作步骤可以看 [网黑哥的文章](https://mp.weixin.qq.com/s/Uv2flkr7_1G0vPSKuKrC1Q)

## 延伸：将书签转为导航站

我们在网上冲浪的时候，发现不错的网站可以 <kbd>CTRL</kbd>​ + <kbd>D</kbd>​ 加入到书签，但书签一多，就不太好管理。

上面提到的几个书签扩展算一个解决办法，但还有另一种：直接将书签导出，做成一个美观且易用的导航网站（例如作为浏览器首页），这样无论你在任何一个浏览器打开网址，都是一样的收藏网址。

而且还可以很方便地分享给他人，还可以通过扩展和浏览器进行同步数据。

​![](https://image.peterjxl.com/blog/image-20250120152841-7axn6vj.png)​
‍

目前市面上有很多这样的项目，我们直接拿来用即可：

* [Start.Me](https://about.start.me/)：专门的书签管理网站，有专门的扩展，可以一键导入浏览器书签、RSS 订阅
* [Webstack](https://github.com/WebStackPage/WebStackPage.github.io)：无后台的纯静态导航
* [OneNav](https://www.onenav.top/)：可与浏览器扩展（插件）配合使用
* [Readeck](https://readeck.org/)：能记录浏览过的网页（支持搜索），支持书签管理（导入/导出，标签）
* [Hoarder](https://hoarder.app/)：基于 AI，可以自动摘录、识别网页，通过 AI 分析内容、生成标签和摘要。
* [mTab](https://www.mtab.cc/)：界面很好看，支持云同步，分类管理书签，无广告
* [Linkwarden](https://linkwarden.app/)：适用于团体协作，单点登录，
* [Pintree](https://www.pintree.io/zh)：界面美观，支持数据导入导出
* [Servas](https://servas.app/)：支持标签，分组，稍后阅读，多用户支持，安全性高
* [Bookmark Pro](https://bookmark-remainder-and-manager-pro.exthub.top/)：革新性的智能书签管理工具，用红黄绿三色信号灯智能提醒，杜绝重复收藏。整合可视化数据分析、历史追踪、智能检测并安全删除重复书签及文件夹、多格式备份等强大功能
* [炫猿导航](https://xydh.fun/)：汇集全网资源大佬的多年收藏，也可以通过它定制自己的个人主页。

上述几个项目，通常会配有 Demo，你可以先在线试试，再考虑是否使用。

而且这些项目基本都开源，支持自部署，可以将数据掌握到自己手中。
