# 1600.这些扩展能让你更好地玩转各大平台

网上冲浪，少不了使用一些大型的论坛和平台，装一些相关扩展也是有必要的。

本文就介绍一些针对大型互联网平台的扩展。

‍<!-- more -->

## B 站

​![](https://image.peterjxl.com/blog/image-20250205152956-d3l0vho.png)​

B 站最早就是以网页起家的，至今它的网页版也仍然非常优秀，所以有很多用户喜欢用。这些用户中，各种开发大佬给 B 站做了很多浏览器扩展，让它的使用体验上升不止一个台阶：

* [BiliPlus](https://github.com/0xlau/biliplus)：作者 UP 主码农小易，每一个功能都直奔痛点，有首页干净模式，首页“换一换”回溯功能，首页干净模式，视频 AI 总结，视频五级倍速，隐藏指定用户评论......
* [BiliScope](https://github.com/gaogaotiantian/biliscope)：作者码农高天，把鼠标放在某个 UP 主的名字上，会弹出一个卡片，让你一键查看 UP 主的投稿信息、个人简介、常用标签，出词云，让你瞬间了解这个 UP；还可以给 UP 添加备注，支持视频 AI 总结
* [BewlyBewly](https://github.com/hakadao/BewlyBewly)：作者 UP 主 Hakadao，几乎是对 B 站页面进行了重构，美化了很多，用了大量毛玻璃、圆角矩形、悬浮菜单的设计，所有功能都被归纳整理，更加简洁、统一、美观。
* [Pakku](https://github.com/xmcp/pakku.js)：弹幕屏蔽插件，比 B 站自带的好用，可以任意设置规则，屏蔽掉刷人数、刷时间等不喜欢或无意义的弹幕
* [ACG 助手](https://acghelper.com)：专为重度 B 站用户定制，全面提升浏览效率，功能超多
* [bilibili 哔哩哔哩下载助手](https://chromewebstore.google.com/detail/bfcbfobhcjbkilcbehlnlchiinokiijp)：B 站功能增强，帮你下载 B 站版权受限（能看不能缓存）的番剧，有着 20w 安装量
* [哔哩哔哩推荐助手](https://chromewebstore.google.com/detail/ebfdljhnmngegkbcllapegggoeopaicl)：可以帮你找和当前 UP 主同个赛道的其他 UP，进一步找到感兴趣的视频
* [B 站空降助手](https://chromewebstore.google.com/detail/eaoelafamejbnggahofapllmfhlhajdd)：带你精准空降到赞助广告结束或者高能时刻，自动跳过视频中的赞助广告、订阅提醒等片段。你也可以亲自标记视频中的广告并上传，所有人都会从您的贡献中受益。
* [弹幕画中画播放器](https://chromewebstore.google.com/detail/nahbabjlllhocabmecfjmcblchhpoclj)：B 站自带的画中画播放很简陋，没弹幕，不能调整进度。用了这个扩展，不但可以看弹幕，还能暂停快进、拖动进度条、调整音量、切换视频，甚至自动给视频生成 CC 字幕和发送弹幕。对比简陋的原版简直好用的不像话。不只支持 B 站，大部分的视频网站都可以用
‍

* [B 站还能装插件？三分钟让你的 B 站像开外挂一样好用！](https://www.bilibili.com/video/av1753315270/)
* [恢复失效收藏夹、无限倍速、超级画中画，这些浏览器插件，让你的 B 站更强大！](https://www.bilibili.com/video/av113474311029315/)
‍
B 站相关的扩展和脚本实在太多，根本说不完。

还有一个 GitHub 项目：https://github.com/HCLonely/awesome-bilibili-extra，收集了很多 B 站相关的扩展 / 脚本 / 程序，有成百上千个，已获得 2.7k Star。

## YouTube

​![](https://image.peterjxl.com/blog/image-20250205153028-ieayd49.png)​

油管，全球最大的视频平台，相关的扩展也很多：

* [画中画 YouTube](https://chromewebstore.google.com/detail/aebidfeikndkapbilgnhbedgilbcngok)：创建一个浮动视频播放器，始终位于所有窗口之上。
* [TubeX](https://chromewebstore.google.com/detail/bifndkhccndcnabjhllngpdapfakfcif)：YouTube AI 免费工具，致力于帮助亚洲用户快速掌握英文信息
* [YouTube 视频速度大师](https://chromewebstore.google.com/detail/nmjcdbffkenhklfchfaekodeimchagga)：提供自动高清和控制视频速度的功能。
* [Dmooji](https://chromewebstore.google.com/detail/dcacgbaadlgfnmcpjncoobionpjnbnih)：YouTube 实时弹幕与互动评论
* [YouTube 中文配音](https://chromewebstore.google.com/detail/oglffgiaiekgeicdgkdlnlkhliajdlja)：自动进行翻译，然后用中文进行朗读
* [Music Mode for YouTube](https://chromewebstore.google.com/detail/abbpaepbpakcpipajigmlpnhlnbennna)：屏蔽视频，只保留 YouTube 和 YouTube Music 的音频。
* [Youtube 中的视频广告拦截器](http://chromewebstore.google.com/detail/gmfgoihkigbggeeliedncoalbhlmgeki)：专门拦截 YouTube 的广告

## V2EX

​![](https://image.peterjxl.com/blog/image-20250204170020-3oqv7iu.png)​

V2EX 里程序员很多，当然也少不了好用的扩展：

[V2EX Polish](https://v2p.app/)：专为 V2EX 用户设计的浏览器插件，集合了众多实用功能，美化界面，展示热门回复，稍后阅读，自动切换主题，评论回复嵌套......

[V2EX Plus](https://chrome.google.com/webstore/detail/v2ex-plus/daeclijmnojoemooblcbfeeceopnkolo)：开源，优雅便捷，支持一键签到与自动签到，新消息通知提醒并改变扩展按钮颜色，只看楼主，插入表情与图片，楼主回复背景色高亮，帖子列表主题预览.......

## 推特

​![](https://image.peterjxl.com/blog/image-20250205152422-zvui55q.png)​

[Control Panel for Twitter](https://chromewebstore.google.com/detail/control-panel-for-twitter/kpmjjdhbcfebfjgdnpjagcndoelnidfj)：精选扩展，200w 用户，增加了缺失的功能和 UI 改进，让你对 Twitter 有更多的控制

[Calm Twitter](https://chromewebstore.google.com/detail/calm-twitter/cknklikacoaeledfaldmhabmldkldocj)：精选扩展，隐藏部分热点信息，让你更专注

[Twitter Lume](https://chromewebstore.google.com/detail/twitter-lume-reskin/khaeenckjiflnchbnejmackcgomcbnmk)：精选扩展，给 Twitter 换个更酷炫的皮肤样式

[X (Twitter) Video Downloader](https://chromewebstore.google.com/detail/x-twitter-video-downloade/akmdionenlnfcipmdhbhcnkighafmdha)：精选扩展，8w 用户，一键下载推特图片和视频（推荐）

## 微博

​![](https://image.peterjxl.com/blog/image-20250205153119-ub595j4.png)​

[眼不见心不烦](https://chrome.google.com/webstore/detail/aognaapdfnnldnjglanfbbklaakbpejm)：还你个干净的微博，提供了各种「宏拉黑」、「过滤话唠」等功能，评分有 4.9 星

[微博扩展](https://chromewebstore.google.com/detail/npegohlaijfkhcjahlgddhanhimeoibn)：支持一键拉黑点赞，批量拉黑，下载视频，备份收藏微博......

[Speechless](https://chromewebstore.google.com/detail/igilfpckopigflpafgoajlljpdhmoall)：把新浪微博的内容，导出成 PDF 文件进行备份，支持时间范围筛选，排版优化

[Weibo 一键下载](https://chromewebstore.google.com/detail/bkgglniclcgkendelemfbgganhojjknh)：一键下载微博视频/图片，是很实用的功能
