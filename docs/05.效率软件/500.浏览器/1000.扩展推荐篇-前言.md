# 100.扩展推荐篇-前言

​![](https://image.peterjxl.com/blog/image-20250117153634-gf7qf0f.png)​

Chrome 的好用除了本身的流畅性以外，80% 的功劳更要归于其强大又顺手的扩展插件。

<!-- more -->

这些插件满足了不同人的不同需求，也将该浏览器的使用体验拉升到了一个「曾今沧海难为水」的境界，几乎你想要的任何功能都可以通过扩展来解决。

接下来几篇文章，就介绍关于扩展的一些技巧，以及一些广受好评的扩展，相信对你会有帮助。

虽然主要介绍的是 Chrome 上的扩展，但只要是同个内核的浏览器，大多数情况下也能兼容。

> 如果你还不知道什么是扩展、如何安装和管理扩展，请参考我之前的文章：《Chrome 扩展全指南》。
‍
## 扩展并不是唯一的解决办法

虽然扩展很多，几乎能做任何事，但并不意味着将一切都交给扩展。例如：

* 一些扩展可以用油猴脚本代替，更轻量
* 一些功能可以用本地软件来解决，例如截图，OCR 等。这样就不仅仅在浏览器内，在任何软件内都能用
* 还可以借助在线网站，例如在线抠图，在线剪辑

咱们装扩展是为了解决问题，提高效率，而不是为了装而装。

如果安装扩展后，发现不好用，或者不符合自己的使用习惯，那么移除扩展也是正常的，不要为了用而用。
‍
## 如何知道自己要装哪些扩展

1. 首先是找能解决自己问题的扩展
2. 其次是看看那些高分好评的扩展，看看自己用不用得上
‍
### 能解决自己问题的扩展

就拿我自己举例，我以前逛 Pixiv 的时候，遇到好看的图片，会自己保存下来。但我保存的办法比较笨，先点开图片，然后鼠标右键 → 图片另存为......

这样其实很麻烦，一来魔法上网速度比较慢，加载图片耗时很久，二来图片多的时候得逐个下载，效率太低太低。

于是，我试着搜索了下关于 Pixiv 的插件，还真发现了一个，它可以直接在图片上方添加一个下载按钮，点击即可在后台下载，还能批量下载，重命名等，大大节省了我的时间。

​![](https://image.peterjxl.com/blog/image-20250117170117-jbycpfv.png)​

举一反三，其他平台也可以找到类似的扩展/脚本，例如 [批量下载 B 站动态页图片](https://www.peterjxl.com/bilibili/dynamic-pic-download/)，[一键下载微博图片和视频](https://www.peterjxl.com/weibo/download/)。
‍
### 去哪找好扩展

网上冲浪的过程中，其实有些操作是不太方便的，但我们已经习惯了，反而觉得没什么问题。

例如，我平时看到某些文库时，里面的内容是不允许复制的。又比如，有些技术论坛得登录后才能复制代码。

之前，我都是打开浏览器控制台，手动复制的，又或者用 OCR。

直到后来发现一个叫「Simple Allow Copy」的扩展，能解除大部分网站不允许复制的限制。

这种平时没注意，但发现后让人眼前一亮的扩展，其实不少，我很多在用的扩展就是这样发现的。

​![](https://image.peterjxl.com/blog/image-20250118152614-xp1vggk.png)​
‍

那么，去哪里找优秀扩展呢？

1. 逛扩展集市，看看哪些扩展是精选的，评分高的，也要注意看看评论
2. 使用搜索技术。之前几篇文章介绍了不少搜索技巧，例如你可以搜「The best Chrome extensions」，还可以细分领域为「Best Chrome extensions for web developers」等等
3. 第三方插件网：之前介绍过如何离线安装扩展，这些网站除了提供下载之外，本身也是个优秀扩展的推荐站，里面有很多文章介绍优秀扩展，还有下载榜单之类的
4. [Chrome 插件英雄榜](https://zhaoolee.gitbooks.io/chrome/content/)：作者试用扩展后，像梁山一百零八好汉排位一样，把一些及其优秀的 Chrome 插件挑选出来做的榜单，配上了图文并茂的中文使用说明，把各方面的设置都做了很诚心的建议
5. [浏览器扩展百宝箱](https://sspai.com/topic/102)：少数派以往推荐的好用的 Chrome 插件

个人建议：偶尔可以逛逛这些网站，或许能有意想不到的收获。
‍
## 注意事项

1. 在安装某个扩展前，注意避雷，例如先搜下有无负面新闻。之前就介绍过，有个超 60 万人在用的扩展 SuperCopy，被 [V2EX 用户曝劫持流量](https://www.v2ex.com/t/1084504)，但目前还未下架。
2. 针对同一个需求，你可能会找到好几个扩展，但都装了可能会引起冲突。可以先停用几个，试试哪个效果好。
3. 安装后，可能需要刷新页面，才能看到扩展生效后的页面
