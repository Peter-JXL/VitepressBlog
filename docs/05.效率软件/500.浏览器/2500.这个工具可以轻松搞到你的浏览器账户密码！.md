## HackBrowserdata

HackBrowserdata，是 Github 上的一个开源项目，它的功能就是从浏览器中解密和导出数据（密码，Cookie、书签、历史记录等），支持 Windows、macOS 和 Linux 三大操作系统。

开源地址：https://github.com/moonD4rk/HackBrowserData，已获得近 12k Star

​![](https://image.peterjxl.com/blog/image-20250214172659-dtnh153.png)​
‍

还可以在官网看到支持的浏览器（不同操作系统下，能破解的浏览器种类也不同，但主流的基本都可以）。

## 使用方法

光说你可能不太信，我来演示下。

首先，在 GitHub 项目的 [release 页面](https://github.com/moonD4rk/HackBrowserData/releases) 中，找到对应的版本并下载。前四个压缩包是 Linux 版，后两个是 Mac 版，然后是 Windows 版：

​![](https://image.peterjxl.com/blog/image-20250214174008-8ojqoh0.png)​
‍

以 Windows 为例，下载后解压，会得到一个 exe 文件，直接运行即可获得结果：

​![](https://image.peterjxl.com/blog/image-20250214174308-3olvjw8.png)​
‍

注意：

* 有些安全软件会认为它是病毒，将其删掉。由于它本质上就是比较敏感的软件，建议将其加入白名单。
* 你可能会不太放行，但它是开源的，一定程度上是可信任的，你甚至可以自己编译它。
‍
运行后，它获取信息的过程非常快，也就 2 ~ 3 秒的时间，过程也会显示在一个黑窗口上：

​![](https://image.peterjxl.com/blog/image-20250214174717-d42e94c.png)​
‍

完成后，它会在当前文件夹下创建目录 results，并将结果都放进去：

​![](https://image.peterjxl.com/blog/image-20250214174530-jrzdvf2.png)​
‍

可以看到，密码、历史记录、书签、下载记录、插件全在里面了。csv 文件可以用 Excel 打开，或者用文本文档。

例如密码，可以看到是以明文显示的（我打码了😶）：

​![](https://image.peterjxl.com/blog/image-20250214174916-5psdfst.png)​
‍

历史记录也是，甚至有访问时间和次数记录：

​![](https://image.peterjxl.com/blog/image-20250214175119-0q9yqwd.png)​
‍

反正是非常厉害了，所有数据都是一览无遗。

## 延伸话题

如果你感觉这个项目有点可怕，那还有更可怕的一个项目：Windows 系统信息查看工具 System Information for Windows，简称 SIM，自称是 Windows 的终极系统信息工具，可以一站式解决、满足对电脑的所有信息需求。

官网：https://www.gtopala.com

这是一个付费软件，价格也不便宜，能试用 14 天。

下载后运行，然后就能查看电脑上几乎所有的数据：

* 硬件信息：CPU，内存，硬盘，电压，温度，PCI......
* 操作系统信息
* 已安装软件信息，版本、安装时间，数据存放目录，使用语言
* 浏览器中的账号密码，甚至支持 IE
* 连过的 WIFI 用户名和密码
* ......

​![](https://image.peterjxl.com/blog/image-20250216115307-8cezetp.png)​
‍

## 最后

这类软件给了我们几个启示：

* 如果是在公众场合使用电脑，建议随时锁屏，并且锁屏密码不能太简单
* 尽量不要外借电脑。如果一定要借，最好在自己眼皮底下进行
* 尽量不要把密码保存到浏览器里，可以用一些更安全的密码管理工具和扩展
* 我在之前的文章《[这些扩展能让 Chrome 原有功能更强！](https://www.peterjxl.com/Browser/Extensions-recommend/oringinal-function-enhance/)》里介绍了自动切换到隐私模式、加密书签的扩展，有需要的同学可以看看。

如果你哪天忘记了密码，不妨来试试这类软件。
