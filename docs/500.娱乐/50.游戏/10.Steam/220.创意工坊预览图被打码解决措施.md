# 预览图模糊解决措施

上一篇文章提到了如何解锁国区限制，之后就可以看到所有内容了。

但如果逛创意工坊，会看到一堆模糊的封面：

‍<!-- more -->

![](https://image.peterjxl.com/blog/image-20241113151420-lrsse79.png)​

为此，有网友写了个油猴脚本：SteamWallpaperDeblur，解决用浏览器逛创意工坊时，部分壁纸缩略图打码的问题。

注意，如果搜不到，可能是因为没有打开 greasyfork 显示成人内容的选项，此时可以：

* 登录 greasyfork → 点击右上角的用户名 → 编辑账户信息 → 勾选显示成人内容的选项，然后再搜一次
* 直接打开页面：https://greasyfork.org/zh-CN/scripts/474291
* 用搜索引擎搜 SteamWallpaperDeblur
* 在 SleazyFork 上搜索
