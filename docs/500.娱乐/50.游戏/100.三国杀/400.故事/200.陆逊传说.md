# 陆逊传说

壮哉神将陆伯言，丈八二桃把营联。

<!-- more -->

一兵队友张文远，二兵袁术只等闲。

三兵大师泪满面，主公孙权笑开颜。

本方残血放万箭，敌人势弱开桃园。

敌方司马与张角，挂上闪电继续连。

铁索全场遭天谴，一道闪电划过天。

小陆求桃二三个，神技自救瞎狗眼。

终于连弩大爆发，队友顿时口无言。

实战经验出真知，八神哪比陆伯言。
