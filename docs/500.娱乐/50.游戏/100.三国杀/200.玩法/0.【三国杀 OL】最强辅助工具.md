## 功能简介

- 日常：

  - 屏蔽动皮、特效
  - 礼包兑换码自动识别，点击即可兑换（不用怕错过兑换码了，也不用打开页面再输入兑换码）
  - 自动完成任务：每日三次免费敲鼓，每日活跃军典任务，开启每日武将免费一抽，领取月卡周卡每日元宝，自动砍元宝树拾取元宝，挂机刷百胜/酒馆亲密度......
  - 皮肤下载，录像下载，自定义背景，屏蔽卡牌特效、击杀动画、战绩胜率统计
  - 支持账号多开、切换
  - 记牌器：记录排名、花色、点数（还可以查询）...
  - 一键观星：一个页面内查看各种奖励和获取方式
  - **在演武模式，还能直接看到各个小金的身份（这个贼好用）**
  - ......
- 山河图：

  - 透视奇遇事件，提前知道对手
  - 每章开局即可开启商店，可以随时进行查看和购买
  - 显示战法的技术：如神龙摆尾、三板斧等
  - .....
- 技能相关：自动计算张昌蒲的严教，糜竺的资援，曹冲的称象，王荣的吉占...

### 桌面微端

最新安装方法：下载一键安装脚本，然后双击安装即可。

‍

以下是旧版本安装方法（即手动安装）：

1. 找到自己电脑上三国杀微端的安装位置， 以默认安装位置 “C:\\Program Files\\SGSOL\\” 为例，打开 resources 文件夹，看到此文件夹内有 “app.asar” 文件，则说明位置正确。

2. 将文件夹内的 “app.asar” 文件重命名为任意其它名称（如改为 “app0.asar”），如果存在名称为 “app” 的文件夹也需要改成其它名称。

3. 点击此处下载修改版微端文件：[app.asar](http://home.ustc.edu.cn//~haoming/sgs/app.asar)，并将下载的“app.asar”移动到刚才改文件名的位置。

4. 最终效果：把原本的 app.asar 替换为下载了的 app.asar。

5. 之后按照以往的方式正常启动三国杀微端，右键开启自动记牌后登录即可生效，或在游戏内点击微端右上角三国杀图标可以显示/隐藏脚本界面。

‍

![](https://blogimg-1258986288.cos.ap-guangzhou.myqcloud.com/blog/image-20241117203330-6go34ev.png)

‍

注意：安装后，很多设置都被还原了，得重新设置下（例如背景音乐，技能自动发动设置，山河图难度等）。

‍

### 网页安装

一、在浏览器中安装 [Tampermonkey（油猴、篡改猴）插件](https://www.tampermonkey.net/)，推荐用 Edge 或 Chrome 浏览器

‍

二、打开篡改猴的脚本管理面板，并点击右上角“实用工具”：

![](https://blogimg-1258986288.cos.ap-guangzhou.myqcloud.com/blog/image-20250423201352-ux8xhgj.png)

‍

如果找不到，可以试着通过链接打开：

- Edge：extension://iikmkjmpaadaobahmlepeloendndfphd/options.html#nav
- Chrome：chrome-extension://dhdgffkkebhmkfjojejmpbldmpobfkfo/options.html#nav=utils

‍

三、在实用工具页面最下方的“从 URL 安装”处，输入链接 http://home.ustc.edu.cn/\~haoming/sgs/script.js，点击并确认安装。

![](https://blogimg-1258986288.cos.ap-guangzhou.myqcloud.com/blog/image-20241117203530-fub7thi.png)

‍

四、重新进入三国杀网页端，脚本即可自动生效。
